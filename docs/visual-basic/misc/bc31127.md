# <a name="addhandler-is-already-declared"></a>"AddHandler" jest już zadeklarowany.
Więcej niż jeden `AddHandler` deklaracji pojawia się w deklaracji zdarzenia niestandardowego. `AddHandler` Deklaracji deklaruje procedurę służącą do dodawania programu obsługi zdarzeń.  
  
 **Identyfikator błędu:** BC31127  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zbędną `AddHandler` instrukcji.
