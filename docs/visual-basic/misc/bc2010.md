# <a name="compilation-failed-message"></a>Kompilacja nie powiodła się: \<wiadomości >
Kompilacja nie powiodła się ze względu na warunkach określonych w komunikacie o błędzie  
  
 **Identyfikator błędu:** BC2010  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź komunikat o błędzie, aby określić dokładny przyczynę błędu.
