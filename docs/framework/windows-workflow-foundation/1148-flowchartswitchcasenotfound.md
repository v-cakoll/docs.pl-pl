# <a name="1148---flowchartswitchcasenotfound"></a>1148 - FlowchartSwitchCaseNotFound
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|1148|  
|Słowa kluczowe|WFActivities|  
|Poziom|Informacje|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 Wskazuje, że można odnaleźć ani uwzględniania wielkości liter lub przypadek domyślny schemat blokowy przełącznika. Wykonywanie działania Flowchart zostanie zakończona.  
  
## <a name="message"></a>Komunikat  
 Schemat blokowy "%1'/FlowSwitch - można odnaleźć ani działania Case ani domyślnego działania Case pasującego do wyniku wyrażenia. Wykonywanie działania Flowchart zostanie zakończona.  
  
## <a name="details"></a>Szczegóły  
  
|Nazwa elementu danych|Typ elementu danych|Opis|  
|--------------------|--------------------|-----------------|  
|Schemat blokowy|xs:String|Nazwa wyświetlana schematu blokowego.|  
|Domeny aplikacji|xs:String|Długość ciągu zwróconego przez AppDomain.CurrentDomain.FriendlyName.|
