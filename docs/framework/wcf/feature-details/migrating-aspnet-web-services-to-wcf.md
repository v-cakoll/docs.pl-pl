# <a name="migrating-aspnet-web-services-to-wcf"></a><span data-ttu-id="a12e4-102">Migrowanie usług sieci Web na platformie ASP.NET do programu WCF</span><span class="sxs-lookup"><span data-stu-id="a12e4-102">Migrating ASP.NET Web Services to WCF</span></span>
<span data-ttu-id="a12e4-103">ASP.NET oferuje .NET Framework biblioteki klas i narzędzia do tworzenia usług sieci Web, a także funkcje usług hostingu w ramach Internet Information Services (IIS).</span><span class="sxs-lookup"><span data-stu-id="a12e4-103">ASP.NET provides .NET Framework class libraries and tools for building Web services, as well as facilities for hosting services within Internet Information Services (IIS).</span></span> <span data-ttu-id="a12e4-104">Windows Communication Foundation (WCF) zapewnia .NET Framework biblioteki klas, narzędzia i usługi hostingu umożliwiające obsługę jednostek oprogramowania w celu komunikowania się przy użyciu dowolnych protokołów, w tym tych używanych przez usługi sieci Web.</span><span class="sxs-lookup"><span data-stu-id="a12e4-104">Windows Communication Foundation (WCF) provides .NET Framework class libraries, tools and hosting facilities for enabling software entities to communicate using any protocols, including those used by Web services.</span></span>  <span data-ttu-id="a12e4-105">Migrowanie usług sieci Web ASP.NET do platformy WCF umożliwia aplikacjom korzystanie z nowych funkcji i ulepszeń, które są unikatowe dla usług WCF.</span><span class="sxs-lookup"><span data-stu-id="a12e4-105">Migrating ASP.NET Web Services to WCF allows your applications to take advantage of new features and improvements that are unique to WCF.</span></span>  
  
 <span data-ttu-id="a12e4-106">Funkcja WCF ma kilka ważnych korzyści względem usług sieci Web ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="a12e4-106">WCF has several important advantages relative to ASP.NET Web services.</span></span> <span data-ttu-id="a12e4-107">Mimo że narzędzia usług sieci Web ASP.NET są przeznaczone wyłącznie do tworzenia usług sieci Web, funkcja WCF udostępnia narzędzia, których można używać w celu komunikacji między jednostkami oprogramowania.</span><span class="sxs-lookup"><span data-stu-id="a12e4-107">While ASP.NET Web services tools are solely for building Web services, WCF provides tools that can be used when software entities must be made to communicate with one another.</span></span> <span data-ttu-id="a12e4-108">Pozwoli to zmniejszyć liczbę technologii, które deweloperzy muszą znać, aby zaspokoić różne scenariusze komunikacji oprogramowania, co z kolei zmniejsza koszty zasobów programistycznych oprogramowania, a także czas na ukończenie projektów programistycznych.</span><span class="sxs-lookup"><span data-stu-id="a12e4-108">This will reduce the number of technologies that developers are required to know in order to accommodate different software communication scenarios, which in turn will reduce the cost of software development resources, as well as the time to complete software development projects.</span></span>  
  
 <span data-ttu-id="a12e4-109">Nawet w przypadku projektów programistycznych usług sieci Web WCF obsługuje więcej protokołów usług sieci Web niż obsługa usług sieci Web ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="a12e4-109">Even for Web service development projects, WCF supports more Web service protocols than ASP.NET Web services support.</span></span> <span data-ttu-id="a12e4-110">Te dodatkowe protokoły zapewniają bardziej zaawansowane rozwiązania obejmujące, między innymi, niezawodne sesje i transakcje.</span><span class="sxs-lookup"><span data-stu-id="a12e4-110">These additional protocols provide for more sophisticated solutions involving, amongst other things, reliable sessions and transactions.</span></span>  
  
 <span data-ttu-id="a12e4-111">Usługa WCF obsługuje więcej protokołów do transportowania komunikatów niż usługi sieci Web ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="a12e4-111">WCF supports more protocols for transporting messages than ASP.NET Web services.</span></span> <span data-ttu-id="a12e4-112">Usługi sieci Web ASP.NET obsługują wysyłanie komunikatów przy użyciu protokołu HTTP (Hypertext Transfer Protocol).</span><span class="sxs-lookup"><span data-stu-id="a12e4-112">ASP.NET Web services only support sending messages by using the Hypertext Transfer Protocol (HTTP).</span></span> <span data-ttu-id="a12e4-113">Usługa WCF obsługuje wysyłanie komunikatów przy użyciu protokołu HTTP, a także Transmission Control Protocol (TCP), nazwane potoki i usługi kolejkowania komunikatów (MSMQ) firmy Microsoft.</span><span class="sxs-lookup"><span data-stu-id="a12e4-113">WCF supports sending messages by using HTTP, as well as the Transmission Control Protocol (TCP), named pipes, and Microsoft Message Queuing (MSMQ).</span></span> <span data-ttu-id="a12e4-114">Ważniejsze, można rozszerzyć obsługę dodatkowych protokołów transportowych.</span><span class="sxs-lookup"><span data-stu-id="a12e4-114">More important, WCF can be extended to support additional transport protocols.</span></span> <span data-ttu-id="a12e4-115">W związku z tym oprogramowanie opracowane przy użyciu programu WCF może zostać dostosowane do pracy z szeroką gamą innego oprogramowania, co zwiększa potencjalne zwroty w odniesieniu do inwestycji.</span><span class="sxs-lookup"><span data-stu-id="a12e4-115">Therefore, software developed using WCF can be adapted to work together with a wider variety of other software, thereby increasing the potential return on the investment.</span></span>  
  
 <span data-ttu-id="a12e4-116">Usługa WCF oferuje wiele bogatszych funkcji wdrażania aplikacji i zarządzania nimi, niż zapewnia usługi sieci Web ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="a12e4-116">WCF provides much richer facilities for deploying and managing applications than ASP.NET Web services provides.</span></span> <span data-ttu-id="a12e4-117">Oprócz systemu konfiguracji, który ASP.NET również ma, usługa WCF oferuje Edytor konfiguracji, śledzenie działań od nadawców do odbiorców oraz z powrotem przez dowolną liczbę pośredników, Podgląd śledzenia, rejestrowanie komunikatów, ogromną liczbę liczników wydajności i obsługę Instrumentacja zarządzania Windows.</span><span class="sxs-lookup"><span data-stu-id="a12e4-117">In addition to a configuration system, which ASP.NET also has, WCF offers a configuration editor, activity tracing from senders to receivers and back through any number of intermediaries, a trace viewer, message logging, a vast number of performance counters, and support for Windows Management Instrumentation.</span></span>  
  
 <span data-ttu-id="a12e4-118">Biorąc pod uwagę te potencjalne korzyści wynikające z usług sieci Web ASP.NET w programie WCF, jeśli używasz programu lub rozważasz korzystanie z usług sieci Web ASP.NET, możesz skorzystać z kilku opcji:</span><span class="sxs-lookup"><span data-stu-id="a12e4-118">Given these potential benefits of WCF relative to ASP.NET Web services, if you are using, or are considering using ASP.NET Web services you have several options:</span></span>  
  
- <span data-ttu-id="a12e4-119">Kontynuuj korzystanie z usług sieci Web ASP.NET i powyżej korzyści proffered przez program WCF.</span><span class="sxs-lookup"><span data-stu-id="a12e4-119">Continue to use ASP.NET Web services, and forego the benefits proffered by WCF.</span></span>  
  
- <span data-ttu-id="a12e4-120">Kontynuuj korzystanie z usług sieci Web ASP.NET z zamiarem zastosowania WCF w pewnym momencie w przyszłości.</span><span class="sxs-lookup"><span data-stu-id="a12e4-120">Keep using ASP.NET Web services with the intention of adopting WCF at some time in the future.</span></span> <span data-ttu-id="a12e4-121">W tematach w tej sekcji wyjaśniono, jak zmaksymalizować potencjalni klienci, aby mogli korzystać z nowych aplikacji usługi sieci Web ASP.NET wraz z przyszłymi aplikacjami WCF.</span><span class="sxs-lookup"><span data-stu-id="a12e4-121">The topics in this section explain how to maximize the prospects for being able to use new ASP.NET Web service applications together with future WCF applications.</span></span> <span data-ttu-id="a12e4-122">W tematach w tej sekcji wyjaśniono również sposób tworzenia nowych usług sieci Web ASP.NET w celu ułatwienia migracji do programu WCF.</span><span class="sxs-lookup"><span data-stu-id="a12e4-122">The topics in this section also explain how to build new ASP.NET Web services so as to make it easier to migrate them to WCF.</span></span> <span data-ttu-id="a12e4-123">Jeśli jednak Zabezpieczanie usług jest ważne lub wymagane są niezawodność lub gwarancje lub jeśli konieczne jest skonstruowanie niestandardowych obiektów zarządzania, jest to lepsza opcja zastosowania funkcji WCF.</span><span class="sxs-lookup"><span data-stu-id="a12e4-123">However, if securing the services is important, or reliability or transaction assurances are required, or if custom management facilities will have to be constructed, then it is a better option to adopt WCF.</span></span> <span data-ttu-id="a12e4-124">Funkcja WCF została zaprojektowana w taki sposób, aby dokładnie takie scenariusze.</span><span class="sxs-lookup"><span data-stu-id="a12e4-124">WCF is designed for precisely such scenarios.</span></span>  
  
- <span data-ttu-id="a12e4-125">Wdrażaj program WCF na potrzeby nowego rozwoju, pozostawiając nadal obsługę istniejących aplikacji usługi sieci Web ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="a12e4-125">Adopt WCF for new development, while continuing to maintain your existing ASP.NET Web service applications.</span></span> <span data-ttu-id="a12e4-126">Ten wybór jest prawdopodobnie optymalny.</span><span class="sxs-lookup"><span data-stu-id="a12e4-126">This choice is very likely the optimal one.</span></span> <span data-ttu-id="a12e4-127">Zapewnia korzyści wynikające z używania programu WCF, ograniczając jednocześnie koszty modyfikacji istniejących aplikacji.</span><span class="sxs-lookup"><span data-stu-id="a12e4-127">It yields the benefits of WCF, while sparing the cost of modifying the existing applications to use it.</span></span> <span data-ttu-id="a12e4-128">W tym scenariuszu nowe aplikacje WCF mogą współistnieć z istniejącymi aplikacjami ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="a12e4-128">In this scenario, new WCF applications can co-exist with existing ASP.NET applications.</span></span> <span data-ttu-id="a12e4-129">Nowe aplikacje WCF będą mogły korzystać z istniejących usług sieci Web ASP.NET, a platforma WCF może służyć do programowania nowych możliwości operacyjnych w istniejących aplikacjach ASP.NET na podstawie trybu zgodności ASP.NET WCF.</span><span class="sxs-lookup"><span data-stu-id="a12e4-129">New WCF applications will be able to use existing ASP.NET Web services, and WCF can be used to program new operational capabilities into existing ASP.NET applications by virtue of WCF ASP.NET compatibility mode.</span></span>  
  
- <span data-ttu-id="a12e4-130">Wdrażaj WCF i Migruj istniejące aplikacje usługi sieci Web ASP.NET do platformy WCF.</span><span class="sxs-lookup"><span data-stu-id="a12e4-130">Adopt WCF and migrate existing ASP.NET Web service applications to WCF.</span></span> <span data-ttu-id="a12e4-131">Możesz wybrać tę opcję, aby ulepszyć istniejące aplikacje przy użyciu funkcji udostępnianych przez program WCF, lub odtworzyć funkcjonalność istniejących usług sieci Web ASP.NET w ramach nowych, wydajnych aplikacji WCF.</span><span class="sxs-lookup"><span data-stu-id="a12e4-131">You may choose this option to enhance the existing applications with features provided by WCF, or to reproduce the functionality of existing ASP.NET Web services within new, more powerful WCF applications.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="a12e4-132">Należy zwrócić uwagę, jeśli usługa WCF jest hostowana przez usługi IIS 5. x i ASP.NET zostanie odinstalowany.</span><span class="sxs-lookup"><span data-stu-id="a12e4-132">Care must be taken if a WCF service is hosted by IIS 5.x and ASP.NET is uninstalled.</span></span> <span data-ttu-id="a12e4-133">Gdy usługa WCF jest hostowana przez program IIS 5. x, kod usługi można zażądać, jeśli ASP.NET zostanie odinstalowany.</span><span class="sxs-lookup"><span data-stu-id="a12e4-133">When a WCF service is hosted by IIS 5.x, the code for the service can be requested if ASP.NET is uninstalled.</span></span> <span data-ttu-id="a12e4-134">Gdy ASP.NET zostanie odinstalowany w systemie operacyjnym, w którym jest uruchomiony program IIS 5. x, a program WCF zostanie odinstalowany, plik z rozszerzeniem svc jest traktowany jako plik tekstowy i zawartość, łącznie z dowolnym kodem źródłowym, jest zwracany do obiektu żądającego.</span><span class="sxs-lookup"><span data-stu-id="a12e4-134">When ASP.NET is uninstalled on an operating system that is running IIS 5.x and WCF is uninstalled, a file with the .svc extension is considered a text file and the contents, including any source code, is returned to the requester.</span></span>
