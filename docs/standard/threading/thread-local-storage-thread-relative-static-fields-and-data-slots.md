- <span data-ttu-id="c6f85-117">Gdy wątek uzyskuje dostęp do tego samego pola lub gniazda z wielu domen aplikacji, w każdej domenie aplikacji jest utrzymywana oddzielna wartość.</span><span class="sxs-lookup"><span data-stu-id="c6f85-117">When a thread accesses the same field or slot from multiple application domains, a separate value is maintained in each application domain.</span></span>
