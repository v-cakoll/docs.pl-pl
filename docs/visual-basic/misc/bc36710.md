# <a name="extension-method-39ltmethodnamegt39-defined-in-39ltmodulenamegt39-does-not-have-a-signature-compatible-with-delegate-39ltdelegatenamegt39"></a>Metody rozszerzenia &#39; &lt;methodname&gt;&#39; zdefiniowanego w &#39;&lt; Nazwa modułu&gt;&#39; nie ma podpisu zgodnego z delegata &#39;&lt; Element delegateName&gt;&#39;
Wystąpiła niezgodność między sygnatury metody rozszerzenia i delegata, który próbujesz użyć. `Delegate` Instrukcji definiuje typy parametrów i zwracanych typów klasa delegata. Postępowanie z zgodnych parametrów, typy parametrów i typ zwracany może służyć do tworzenia wystąpienia tego typu obiektu delegowanego.
