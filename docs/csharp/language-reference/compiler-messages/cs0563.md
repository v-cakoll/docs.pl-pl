# <a name="compiler-error-cs0563"></a>Błąd kompilatora CS0563
Jeden z parametrów operatora binarnego musi być typem zawierającym  
  
Deklaracja metody dla [przeciążenia operatora](../keywords/operator.md) musi być zgodna z zasadami.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0563:  
  
```csharp  
// CS0563.cs  
public class iii  
{  
    public static implicit operator int(iii x)  
    {  
        return 0;  
    }  
    public static implicit operator iii(int x)  
    {  
        return null;  
    }  
    public static int operator +(int aa, int bb)   // CS0563   
    // Use the following line instead:  
    // public static int operator +(int aa, iii bb)      
    {  
        return 0;  
    }  
    public static void Main()  
    {  
    }  
}  
```
