# <a name="shared-39sub-new39-cannot-have-any-parameters"></a>Udostępnione &#39;Sub New&#39; nie może mieć żadnych parametrów
Parametr lub parametry zostały uwzględnione w udostępnionej `Sub New` instrukcji, która jest nieprawidłowa.  
  
 **Identyfikator błędu:** BC30479  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń parametr lub parametrów.
