# <a name="compiler-error-cs2007"></a>CS2007 błąd kompilatora
Nierozpoznana opcja wiersza polecenia: "option"  
  
 Kompilator został przekazany ciąg, który nie był [— opcja kompilatora](../../csharp/language-reference/compiler-options/index.md), nawet jeśli rozpoczęto od ukośnika (/).  
  
 Poniższy przykład generuje CS2007:  
  
```csharp  
// CS2007.cs  
// compile with: /recur  
// CS2007 expected  
class x  
{  
   public static void Main() {}  
}  
```
