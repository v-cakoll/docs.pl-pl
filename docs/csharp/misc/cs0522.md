# <a name="compiler-error-cs0522"></a>Błąd kompilatora CS0522
"Konstruktor": struktury nie mogą wywoływać konstruktorów klasy bazowej  
  
 [Struktura](../language-reference/keywords/struct.md) nie może wywołać konstruktora klasy bazowej; Usuń wywołanie do konstruktora klasy bazowej.  
  
 Poniższy przykład generuje CS0522:  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
