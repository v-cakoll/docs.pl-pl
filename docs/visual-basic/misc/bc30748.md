# <a name="cannot-convert-to-type"></a>Nie można przekonwertować na " \<type> "
Podjęto próbę wykonania nieprawidłowej konwersji.  
  
 **Identyfikator błędu:** BC30748  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Wybierz zgodny typ konwersji.
