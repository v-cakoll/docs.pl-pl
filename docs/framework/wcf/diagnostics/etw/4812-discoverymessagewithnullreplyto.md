# <a name="4812---discoverymessagewithnullreplyto"></a>4812 — DiscoveryMessageWithNullReplyTo
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|Identyfikator|4812|  
|słowa kluczowe|Odnajdywanie|  
|Poziom|Ostrzeżenie|  
|Kanał|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Opis  
 To zdarzenie jest emitowane, gdy komunikat odnajdowanie zostało porzucone, klasa DiscoveryClient, ponieważ nie ma adresu ReplyTo.  
  
## <a name="message"></a>Komunikat  
 Komunikat żądania odnajdywania z messageId = "%1" został porzucony, ponieważ nie ma adresu ReplyTo.
