# <a name="compiler-error-cs0431"></a>Błąd kompilatora CS0431
Nie można użyć aliasu "identifier" z '::' ponieważ alias odwołuje się do typu. Zamiast tego użyj ".'.  
  
 Użyto "::" z aliasem, który odwołuje się do typu. Aby rozwiązać ten błąd, należy użyć operatora ".".  
  
 Poniższy przykład generuje CS0431:  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer
{  
   public class Inner
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
