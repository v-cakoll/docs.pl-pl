# <a name="compiler-warning-level-1-cs3001"></a>Kompilator CS3001 ostrzegawcze (poziom 1)
Typ argumentu "type" nie jest zgodne ze specyfikacją CLS  
  
 A [publicznego](../../csharp/language-reference/keywords/public.md), [chronione](../../csharp/language-reference/keywords/protected.md), lub `protected internal` metoda musi obsługiwać parametru, którego typ jest zgodne z typowych specyfikacji języka (CLS). Aby uzyskać więcej informacji dotyczących zgodności ze specyfikacją CLS, zobacz [pisanie kodu zgodne ze specyfikacją CLS](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3) i [niezależność od języka i elementy niezależne od języka](../../../docs/standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS3001:  
  
```csharp  
// CS3001.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public void bad(ushort i)   // CS3001  
    {  
    }  
  
    private void OK(ushort i)   // OK, private method  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
