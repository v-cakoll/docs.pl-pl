# <a name="unable-to-load-information-for-class-39ltclassnamegt39"></a>Nie można załadować informacji o klasie &#39; &lt;classname&gt;&#39;
Odwołanie zostało przesłane do klasy, która nie jest dostępna.  
  
 **Identyfikator błędu:** BC30712  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy ta klasa jest zdefiniowana i prawidłowo wpisana nazwa.  
  
2.  Spróbuj, uzyskiwanie dostępu do jednego z elementów członkowskich zadeklarowany w module. W niektórych przypadkach debugowania środowiska nie można zlokalizować elementów członkowskich, ponieważ moduły, w którym je zadeklarowano nie zostały jeszcze załadowane.
