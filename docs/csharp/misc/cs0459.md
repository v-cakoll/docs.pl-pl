# <a name="compiler-error-cs0459"></a><span data-ttu-id="70de1-102">CS0459 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="70de1-102">Compiler Error CS0459</span></span>
<span data-ttu-id="70de1-103">Nie można przyjąć adresu zmiennej lokalnej tylko do odczytu</span><span class="sxs-lookup"><span data-stu-id="70de1-103">Cannot take the address of a read-only local variable</span></span>  
  
 <span data-ttu-id="70de1-104">Istnieją trzy typowe scenariusze w języku C# generujących zmiennych lokalnych tylko do odczytu: `foreach`, `using`, i `fixed`.</span><span class="sxs-lookup"><span data-stu-id="70de1-104">There are three common scenarios in the C# language that generate read-only local variables: `foreach`, `using`, and `fixed`.</span></span> <span data-ttu-id="70de1-105">W każdym z tych przypadków nie są dozwolone zapisu do zmiennej lokalnej tylko do odczytu lub pobrać jej adresu.</span><span class="sxs-lookup"><span data-stu-id="70de1-105">In each of these cases, you are not allowed to write to the read-only local variable, or to take its address.</span></span> <span data-ttu-id="70de1-106">Ten błąd jest generowany, gdy kompilator realizuje się, że próbujesz uzyskać adresu zmiennej lokalnej tylko do odczytu.</span><span class="sxs-lookup"><span data-stu-id="70de1-106">This error is generated when the compiler realizes you are trying to take the address of a read-only local variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="70de1-107">Przykład</span><span class="sxs-lookup"><span data-stu-id="70de1-107">Example</span></span>  
 <span data-ttu-id="70de1-108">Podczas próby przyjąć adresu zmiennej lokalnej tylko do odczytu w poniższym przykładzie generuje CS0459 `foreach` pętli i w `fixed` blok instrukcji.</span><span class="sxs-lookup"><span data-stu-id="70de1-108">The following example generates CS0459 when an attempt is made to take the address of a read-only local variable in a `foreach` loop and in a `fixed` statement block.</span></span>  
  
```csharp  
// CS0459.cs  
// compile with: /unsafe  
  
class A  
{  
    public unsafe void M1()  
    {  
        int[] ints = new int[] { 1, 2, 3 };  
        foreach (int i in ints)  
        {  
            int *j = &i;  // CS0459  
        }  
  
        fixed (int *i = &_i)  
        {  
            int **j = &i;  // CS0459  
        }  
    }  
  
    private int _i = 0;  
}  
```
