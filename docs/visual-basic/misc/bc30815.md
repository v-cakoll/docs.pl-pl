# <a name="functionname-is-not-declared-visual-basic-error"></a>'\<functionname >' nie został zadeklarowany (błąd w Visual Basic)
'\<functionname >' nie został zadeklarowany. Funkcje We/Wy pliku jest dostępna w przestrzeni nazw Microsoft.VisualBasic.  
  
 **Identyfikator błędu:** BC30815  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Użyj funkcji we/wy pliku zdefiniowanych w `My.Computer.FileSystem`, `System.IO` funkcje środowiska wykonawczego języka Visual Basic, przestrzeni nazw lub `Microsoft.VisualBasic.FileSystem` obiektu.
