# <a name="is-expected"></a>Oczekiwano znaku "is"
`TypeOf`Operator występuje bez `Is` klauzuli.  
  
 **Identyfikator błędu:** BC30224  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj `Is` klauzulę po `TypeOf` operatorze; na przykład `TypeOf MyControl Is ComboBox` .
