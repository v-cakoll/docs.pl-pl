# <a name="39lttypenamegt39-must-be-declared-39mustinherit39-because-it-contains-methods-declared-39mustoverride39"></a>&#39; &lt;typename&gt;&#39; musi być zadeklarowany jako &#39; MustInherit &#39; ponieważ zawiera metody zadeklarowane jako &#39; MustOverride &#39;
Typy, które zawierają `MustOverride` musi być zadeklarowany jako elementy członkowskie `MustInherit`.  
  
 **Identyfikator błędu:** BC31411  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `MustInherit` modyfikator do typu.
