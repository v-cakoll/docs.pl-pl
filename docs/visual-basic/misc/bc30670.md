# <a name="39redim39-statements-require-a-parenthesized-list-of-the-new-bounds-of-each-dimension-of-the-array"></a>&#39; ReDim &#39; instrukcje wymaga ujętej w nawiasy listy nowych granic dla każdego wymiaru tablicy
Należy określić nowy rozmiar tablicy jako część `ReDim` instrukcji.  
  
 **Identyfikator błędu:** BC30670  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj rozmiar każdego rangą tablicy; na przykład:  
  
    ```  
    ReDim arr(5, 6)  
    ```
