# <a name="public-member-39ltmembernamegt39-on-type-39lttypenamegt39-not-found"></a>Publicznego elementu członkowskiego &#39; &lt;membername&gt;&#39; na typ &#39;&lt; Właściwość TypeName&gt;&#39; nie znaleziono
Nie można odnaleźć członka, którego próbujesz uzyskać dostęp. Ten błąd może wystąpić podczas uzyskiwania dostępu do elementów członkowskich niepublicznych obiektów z późnym wiązaniem.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Jeśli istnieje element członkowski, upewnij się, że jest on zadeklarowany jako `Public`.
