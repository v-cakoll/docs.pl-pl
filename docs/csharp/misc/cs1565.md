# <a name="compiler-error-cs1565"></a>CS1565 błąd kompilatora

 Nie jest prawidłową określić ich obu [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) — opcja kompilatora i [/win32ico](../../csharp/language-reference/compiler-options/win32icon-compiler-option.md) — opcja kompilatora w tej samej kompilacji.
