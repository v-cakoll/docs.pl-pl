# <a name="isnot-requires-operands-that-have-reference-types-but-this-operand-has-the-value-type-typename"></a>'IsNot' wymaga argumentów operacji, które mają typ referencyjny, ale ten argument operacji ma typ wartości '\<typename >'.
Wyrażenie używa [operatora IsNot](../../visual-basic/language-reference/operators/isnot-operator.md) z co najmniej jednym argumentem typu wartości.  
  
 Operator `IsNot` określa, czy dwa odwołania do obiektów odwołują się do różnych obiektów. Porównuje wartości wskaźnika typów referencyjnych i nie ma znaczenia z typami wartości.  
  
 **Identyfikator błędu:** BC31419  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Jeśli zamierzasz porównać wartości dwóch typów wartości, użyj `=` operatora porównania lub. `<>`  
  
- Jeśli zamierzasz porównać wskaźniki dwóch typów referencyjnych, upewnij się, że używasz odwołań do obiektów dla obu operandów. Możesz użyć zmiennych lub elementów odniesienia, takich jak słowo kluczowe [Me](../programming-guide/program-structure/me-my-mybase-and-myclass.md#me) , które zachowują się jak zmienne odwołania.
