# <a name="compiler-error-cs0500"></a>CS0500 błąd kompilatora
"składowej klasy" nie można zadeklarować treść, ponieważ jest oznaczony jako abstrakcyjny  
  
 [Abstrakcyjne](../../csharp/language-reference/keywords/abstract.md) metoda nie może zawierać jego wykonania.  
  
 Poniższy przykład spowoduje wygenerowanie CS0500:  
  
```csharp  
// CS0500.cs  
namespace x  
{  
   abstract public class clx  
   {  
      abstract public void f(){}   // CS0500  
      // try the following line instead  
      // abstract public void f();  
   }  
  
   public class cly  
   {  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
