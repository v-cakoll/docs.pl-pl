# <a name="expected-matching-closing-single-quote-for-xml-attribute-value"></a>Oczekiwano dopasowania zamykającego pojedynczego cudzysłowu dla wartości atrybutu XML
Wartość atrybutu, która jest rozdzielana pojedynczym cudzysłowem, nie zawiera znaku cudzysłowu zamykającego.  
  
 **Identyfikator błędu:** BC31163  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj znak cudzysłowu zamykającego apostrofu ('), aby rozdzielić wartość atrybutu.
