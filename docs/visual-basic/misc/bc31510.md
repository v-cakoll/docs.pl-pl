# <a name="attribute-constant-39ltconstantnamegt39-cannot-be-the-target-of-an-assignment"></a>Stała atrybutu &#39; &lt;constantname&gt; &#39; nie może być elementem docelowym przypisania
Nastąpiła próba do przypisania wartości stałej zadeklarowanego w atrybucie.  
  
 **Identyfikator błędu:** BC31510  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń przypisanie błędne.  
  
-   Jeśli używasz niestandardowych atrybutów, które utworzono ponownie zdefiniować element członkowski w atrybucie jako pole.
