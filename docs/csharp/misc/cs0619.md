# <a name="compiler-error-cs0619"></a><span data-ttu-id="942fe-102">CS0619 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="942fe-102">Compiler Error CS0619</span></span>
<span data-ttu-id="942fe-103">"członek" jest przestarzały: "tekst"</span><span class="sxs-lookup"><span data-stu-id="942fe-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="942fe-104">Element członkowski klasy został oznaczony za pomocą [przestarzałe] ((.. /.. / csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) atrybutu w taki sposób, że gdy odwołuje się do składowej klasy zostanie wygenerowany błąd.</span><span class="sxs-lookup"><span data-stu-id="942fe-104">A class member was marked with the [Obsolete]((../../csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="942fe-105">Poniższy przykład spowoduje wygenerowanie CS0619:</span><span class="sxs-lookup"><span data-stu-id="942fe-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
