# <a name="39ltinterfacename1gt39-cannot-implement-39ltmethodnamegt39-because-there-is-no-matching-ltmethodgt-on-interface-39ltinterfacename2gt39"></a>&#39; &lt;interfacename1&gt;&#39; nie implementuje &#39;&lt; methodname&gt;&#39; ponieważ nie ma pasującego &lt;metody&gt; na interfejs &#39;&lt; interfacename2&gt;&#39;
Nastąpiła próba zaimplementować metodę w interfejsie, gdy w interfejsie, z której zostały uzyskane pierwszy interfejs nie istnieje metoda dopasowania.  
  
 **Identyfikator błędu:** BC30401  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że nie ma została błędnie wpisana nazwa metody, lub podaj brakującej metody.
