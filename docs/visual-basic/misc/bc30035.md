# <a name="syntax-error-visual-basic-error"></a>Błąd składniowy (Błąd Visual Basic)
Wiersz kodu źródłowego generuje błąd składniowy nieokreślony.  
  
 **Identyfikator błędu:** BC30035  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź dokumentację dla każdego słowa kluczowe użyte w wierszu kodu źródłowego. Porównaj przykładowy kod w dokumentacji względem wiersza źródłowego, generowanie tego błędu.  
  
2.  Jeśli nie można zidentyfikować przyczynę błędu, zebrać informacje dotyczące okoliczności i powiadomić pomocy technicznej firmy Microsoft.
