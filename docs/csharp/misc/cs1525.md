# <a name="compiler-error-cs1525"></a>CS1525 błąd kompilatora
Nieprawidłowe wyrażenie termin "character"  
  
 Kompilator wykrył nieprawidłowy znak w wyrażeniu.  
  
 Poniższy przykład generuje CS1525:  
  
```csharp  
// CS1525.cs  
class x  
{  
   public static void Main()  
   {  
      int i = 0;  
      i = i +   // OK - identifier  
      'c' +     // OK - character  
      (5) +     // OK - parenthesis  
      [ +       // CS1525, operator not a valid expression element  
      throw +   // CS1525, keyword not allowed in expression  
      void;     // CS1525, void not allowed in expression  
   }  
}  
```  
  
 Puste etykiety można również generować CS1525, jak w poniższym przykładzie:  
  
```csharp  
// CS1525b.cs  
using System;  
public class MyClass  
{  
   public static void Main()  
   {  
      goto FoundIt;  
      FoundIt:      // CS1525  
      // Uncomment the following line to resolve:  
      // System.Console.Write("Hello");  
   }  
}  
```
