# <a name="39end-interface39-must-be-preceded-by-a-matching-39interface39"></a>&#39; Interfejs &#39; musi być poprzedzona odpowiadającą jej instrukcją &#39; interfejs &#39;
`End Interface` Występuje instrukcja bez odpowiadającego `Interface` instrukcji. `End Interface`musi być poprzedzona odpowiadającą `Interface` instrukcji.  
  
 **Identyfikator błędu:** BC30252  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź, czy występuje przed `Interface` instrukcja jest nieprawidłowo zapisana lub są one nieprawidłowe.  
  
2.  Upewnij się, że `Interface` elementy członkowskie są prawidłowo sformatowane.
