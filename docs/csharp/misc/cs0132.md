# <a name="compiler-error-cs0132"></a>CS0132 błąd kompilatora
"Konstruktor": Konstruktor statyczny nie może mieć parametrów  
  
 A [statyczne](../../csharp/language-reference/keywords/static.md) nie można zadeklarować konstruktora z co najmniej jeden parametr. Aby uzyskać więcej informacji, zobacz [konstruktory](../../csharp/programming-guide/classes-and-structs/constructors.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0132:  
  
```csharp  
// CS0132.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public MyClass(int i)  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass  
   {  
      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
