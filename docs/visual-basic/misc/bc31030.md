# <a name="project-level-conditional-compilation-constant-constantname-is-not-valid-error"></a>Stała kompilacji warunkowej na poziomie projektu "\<constantname >" jest nieprawidłowa: \<błąd >
Opcja nieprawidłowy kompilacja warunkowa jest określona dla projektu.  
  
 **Identyfikator błędu:** BC31030  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź stałe kompilacji warunkowej, używane z tym projektem.
