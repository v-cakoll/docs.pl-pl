# <a name="compiler-error-cs0653"></a>CS0653 błąd kompilatora
Nie można zastosować klasy atrybutów "class", ponieważ jest ona abstrakcyjna  
  
 [Abstrakcyjny](../../csharp/language-reference/keywords/abstract.md) klasy atrybutów niestandardowych nie można użyć jako atrybut.  
  
 Poniższy przykład generuje CS0653:  
  
```csharp  
// CS0653.cs  
using System;  
  
public abstract class MyAttribute : Attribute  
{  
}  
  
public class My2Attribute : MyAttribute  
{  
}  
  
[My]   // CS0653  
// try the following line instead  
// [My2]  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
