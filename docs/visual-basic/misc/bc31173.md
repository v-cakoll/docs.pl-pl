# <a name="missing-required-white-space"></a>Brak wymaganego odstępu.
Literał XML został określony bez spacji rozdzielające. Przykład miejsca rozdzielające to odstęp między kolejnych atrybutów w elemencie XML literału.  
  
 **Identyfikator błędu:** BC31173  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Dodaj wymagane miejsce separatora.
