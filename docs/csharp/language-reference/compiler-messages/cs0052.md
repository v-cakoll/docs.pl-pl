# <a name="compiler-error-cs0052"></a>Błąd kompilatora CS0052

Niespójna dostępność: typ pola jest mniej dostępny niż pole "pole"

 Typ pola nie może być mniej dostępne niż samo pole, ponieważ wszystkie konstrukcje publiczne musi zwracać publicznie dostępny obiekt.

## <a name="example"></a>Przykład

 Poniższy przykład generuje CS0052:

```csharp
// CS0052.cs
public class MyClass2
{
    // The following line causes an error because the field, M, is declared
    // as public, but the type, MyClass, is private. Therefore the type is
    // less accessible than the field.
    public MyClass M;   // CS0052

    private class MyClass
    {
    }
    // One way to resolve the error is to change the accessibility of the type
    // to public.
    //public class MyClass
    // Another solution is to change the accessibility of the field to private.
    // private MyClass M;
}

public class MainClass
{
    public static void Main()
    {
    }
}
```
