# <a name="compiler-error-cs0175"></a><span data-ttu-id="535fe-102">Błąd kompilatora CS0175</span><span class="sxs-lookup"><span data-stu-id="535fe-102">Compiler Error CS0175</span></span>
<span data-ttu-id="535fe-103">Użycie słowa kluczowego "Base" jest nieprawidłowe w tym kontekście</span><span class="sxs-lookup"><span data-stu-id="535fe-103">Use of keyword 'base' is not valid in this context</span></span>  
  
 <span data-ttu-id="535fe-104">[Podstawowe](../language-reference/keywords/base.md) słowo kluczowe musi być używane do określania określonego elementu członkowskiego klasy podstawowej.</span><span class="sxs-lookup"><span data-stu-id="535fe-104">The [base](../language-reference/keywords/base.md) keyword must be used to specify a particular member of the base class.</span></span> <span data-ttu-id="535fe-105">Aby uzyskać więcej informacji, [](../programming-guide/classes-and-structs/constructors.md)Zobacz konstruktory.</span><span class="sxs-lookup"><span data-stu-id="535fe-105">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="535fe-106">Poniższy przykład generuje CS0175:</span><span class="sxs-lookup"><span data-stu-id="535fe-106">The following sample generates CS0175:</span></span>  
  
```csharp  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
