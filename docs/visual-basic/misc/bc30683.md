# <a name="inherits-statement-must-precede-all-declarations-in-a-class"></a>Instrukcja "Inherits" musi poprzedzać wszystkie deklaracje w klasie
`Inherits` Instrukcje muszą znajdować się na początku definicji klas.  
  
 **Identyfikator błędu:** BC30683  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- `Inherits` Przenieś instrukcję na początek klasy.
