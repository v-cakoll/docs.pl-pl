# <a name="compiler-error-cs1536"></a>CS1536 błąd kompilatora
Nieprawidłowy parametr typu void  
  
 Nie jest konieczne, lub można określić [void](../../csharp/language-reference/keywords/void.md) parametrów innych niż wskaźnika void.  
  
 Poniższy przykład spowoduje wygenerowanie CS1536:  
  
```csharp  
// CS1536.cs  
class a  
{  
   public static int x( void )   // CS1536  
   // try the following line instead  
   // public static int x()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
