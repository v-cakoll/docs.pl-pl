# <a name="compiler-error-cs0254"></a><span data-ttu-id="08896-102">Compiler Error CS0254</span><span class="sxs-lookup"><span data-stu-id="08896-102">Compiler Error CS0254</span></span>
<span data-ttu-id="08896-103">Po prawej stronie przypisania fixed — instrukcja nie może być wyrażeniem rzutowania</span><span class="sxs-lookup"><span data-stu-id="08896-103">The right hand side of a fixed statement assignment may not be a cast expression</span></span>  
  
 <span data-ttu-id="08896-104">Po prawej stronie [stałej](../../csharp/language-reference/keywords/fixed-statement.md) wyrażenia nie może używać rzutowania.</span><span class="sxs-lookup"><span data-stu-id="08896-104">The right side of a [fixed](../../csharp/language-reference/keywords/fixed-statement.md) expression may not use a cast.</span></span> <span data-ttu-id="08896-105">Aby uzyskać więcej informacji, zobacz [niebezpieczny kod i wskaźniki](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="08896-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="08896-106">Poniższy przykład spowoduje wygenerowanie CS0254:</span><span class="sxs-lookup"><span data-stu-id="08896-106">The following sample generates CS0254:</span></span>  
  
```csharp  
// CS0254.cs  
// compile with: /unsafe  
class Point  
{  
   public uint x, y;  
}  
  
class FixedTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      *p *= *p;  
   }  
  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
      pt.x = 5;  
      pt.y = 6;  
  
      fixed (int* p = (int*)&pt.x)   // CS0254  
      // try the following line instead  
      // fixed (uint* p = &pt.x)  
      {  
         SquarePtrParam ((int*)p);  
      }  
   }  
}  
```
