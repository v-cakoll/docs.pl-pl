# <a name="variable-cannot-be-initialized-with-non-array-type-elementname"></a>Nie można zainicjować zmiennej przy użyciu typu nietablicowego '\<elementname >'
Zmienna, która jest zadeklarowana jako tablica musi zostać zainicjowany z wartości tablicy.  
  
```  
' Not valid.  
' The following line causes this error when executed with Option Strict off.  
' Dim arrayVar1() = 10  
```  
  
 **Identyfikator błędu:** BC36536  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Inicjuj zmienną tablicy przy użyciu wartości tablicy:  
  
    ```
    ' With Option Strict off.  
    Dim arrayVar2() = {1, 2, 3}  
    ' With Option Strict on.  
    Dim arrayVar3() As Integer = {1, 2, 3}  
    ```
