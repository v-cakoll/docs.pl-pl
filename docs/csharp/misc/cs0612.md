# <a name="compiler-warning-level-1-cs0612"></a>Kompilatora (poziom 1) ostrzeżenie CS0612
"członek" jest przestarzały  
  
 Projektant klas oznaczona jako element członkowski o [Obsolete](https://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) atrybutu. Oznacza to, elementu członkowskiego nie mogą być obsługiwane w przyszłych wersjach klasy.  
  
 Poniższy przykład pokazuje, jak uzyskiwanie dostępu do przestarzała składowa generuje CS0612:  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
