# <a name="compiler-error-cs0004"></a>CS0004 błąd kompilatora
Ostrzeżenie traktowane jako błąd  
  
 Skompilowane z `/warnaserror` opcję kompilatora, która umożliwia kompilatorowi generowanie ostrzeżenia jako błędy.
