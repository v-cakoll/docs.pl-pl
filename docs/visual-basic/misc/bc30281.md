# <a name="structure-39ltstructurenamegt39-must-contain-at-least-one-instance-member-variable-or-event-declaration"></a>Struktura &#39; &lt;structurename&gt; &#39; musi zawierać co najmniej jedno wystąpienie zmiennej członkowskiej lub deklaracja zdarzenia
Występuje funkcjonalnie pustą strukturą. Co najmniej jednego członka zmiennej musi być zadeklarowany jako między `Structure` instrukcji i `End Structure` instrukcji.  
  
 **Identyfikator błędu:** BC30281  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `Dim` instrukcji lub `Event` instrukcji w strukturze.
