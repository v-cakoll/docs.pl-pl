# <a name="icordebugevalgetthread-method"></a>ICorDebugEval::GetThread — Metoda
Pobiera wątku, w którym tej oceny jest wykonywany lub będą wykonywane.  
  
## <a name="syntax"></a>Składnia  
  
```  
HRESULT GetThread (  
    [out] ICorDebugThread   **ppThread  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `ppThread`  
 [out] Wskaźnik do adresu ICorDebugThread obiekt, który reprezentuje wątku.
