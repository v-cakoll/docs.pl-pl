# <a name="type-cannot-be-inherited-more-than-once"></a>\<type>nie można dziedziczyć "" więcej niż raz
Zduplikowano `Inherits` instrukcję.  
  
 **Identyfikator błędu:** BC30584  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń instrukcję nadmiarową `Inherits` .
