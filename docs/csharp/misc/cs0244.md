# <a name="compiler-error-cs0244"></a>Błąd kompilatora CS0244
Żadne "is" ani "As" nie są prawidłowe w typach wskaźników  
  
 Operatory [is](../../csharp/language-reference/operators/type-testing-and-cast.md#is-operator) i [as](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) są nieprawidłowe do użycia w typach wskaźników. Aby uzyskać więcej informacji, zobacz artykuł [niebezpieczny kod i wskaźniki](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Poniższy przykład generuje CS0244:  
  
```csharp  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
