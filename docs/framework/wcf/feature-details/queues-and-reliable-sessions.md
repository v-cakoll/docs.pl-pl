## <a name="reliable-sessions"></a><span data-ttu-id="643e1-114">Niezawodne sesje</span><span class="sxs-lookup"><span data-stu-id="643e1-114">Reliable Sessions</span></span>

## <a name="queues"></a><span data-ttu-id="643e1-121">Kolejki</span><span class="sxs-lookup"><span data-stu-id="643e1-121">Queues</span></span>  
 <span data-ttu-id="643e1-122">Kolejki w programie WCF zapewniają niezawodne transfery komunikatów i separację między źródłami i miejscami docelowymi kosztem dużego opóźnienia.</span><span class="sxs-lookup"><span data-stu-id="643e1-122">Queues in WCF provide both reliable transfers of messages and separation between sources and destinations at the cost of high latency.</span></span> <span data-ttu-id="643e1-123">Komunikacja w kolejce WCF jest oparta na usłudze kolejkowania komunikatów (znanej także jako MSMQ).</span><span class="sxs-lookup"><span data-stu-id="643e1-123">WCF queued communication is built on top of Message Queuing (also known as MSMQ).</span></span>  
  
 <span data-ttu-id="643e1-124">Usługa MSMQ jest dostarczana jako opcja systemu Windows, która działa jako usługa NT.</span><span class="sxs-lookup"><span data-stu-id="643e1-124">MSMQ is shipped as an option with Windows that runs as an NT service.</span></span> <span data-ttu-id="643e1-125">Przechwytuje komunikaty do transmisji w kolejce transmisji w imieniu źródła i dostarcza je do kolejki docelowej.</span><span class="sxs-lookup"><span data-stu-id="643e1-125">It captures messages for transmission in a transmission queue on behalf of the source and delivers it to a target queue.</span></span> <span data-ttu-id="643e1-126">Kolejka docelowa akceptuje komunikaty w imieniu miejsca docelowego w celu późniejszego dostarczenia za każdym razem, gdy docelowe żądania komunikatów.</span><span class="sxs-lookup"><span data-stu-id="643e1-126">The target queue accepts messages on behalf of the destination for later delivery whenever the destination requests for messages.</span></span> <span data-ttu-id="643e1-127">Menedżerowie kolejki MSMQ implementują niezawodny protokół transferu komunikatów, dzięki czemu komunikaty nie są tracone w transmisji.</span><span class="sxs-lookup"><span data-stu-id="643e1-127">The MSMQ queue managers implement a reliable message-transfer protocol so that messages are not lost in transmission.</span></span> <span data-ttu-id="643e1-128">Protokół może być natywny lub oparty na protokole SOAP, taki jak protokół SOAP unmessaging Protocol (SRMP).</span><span class="sxs-lookup"><span data-stu-id="643e1-128">The protocol can be native or SOAP-based, such as Soap Reliable Messaging Protocol (SRMP).</span></span>  
  
 <span data-ttu-id="643e1-129">Separacja, w połączeniu z niezawodnymi transferami komunikatów między kolejkami, umożliwia niezawodną komunikację z aplikacjami, które są luźno połączone.</span><span class="sxs-lookup"><span data-stu-id="643e1-129">The separation, coupled with reliable message transfers between queues, enables applications that are loosely coupled to communicate reliably.</span></span> <span data-ttu-id="643e1-130">W przeciwieństwie do sesji niezawodnych, źródło i miejsce docelowe nie muszą być uruchomione w tym samym czasie.</span><span class="sxs-lookup"><span data-stu-id="643e1-130">Unlike reliable sessions, the source and destination do not have to be running at the same time.</span></span> <span data-ttu-id="643e1-131">To niejawnie włącza scenariusze, w których kolejki są stosowane jako mechanizm poziomu obciążenia, gdy występuje niezgodność między szybkością produkcji komunikatów przez źródło i szybkością zużycia komunikatów przez miejsce docelowe.</span><span class="sxs-lookup"><span data-stu-id="643e1-131">This implicitly enables scenarios where queues are, in effect, used as a load-leveling mechanism when there is a mismatch between the rate of message production by the source and the rate of the message consumption by the destination.</span></span> <span data-ttu-id="643e1-132">Aby uzyskać więcej informacji o kolejkach, zobacz [kolejki w programie WCF](queues-in-wcf.md).</span><span class="sxs-lookup"><span data-stu-id="643e1-132">For more information about queues, see [Queues in WCF](queues-in-wcf.md).</span></span>
