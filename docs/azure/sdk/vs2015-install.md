# <a name="azure-tools-for-visual-studio-2015"></a>Narzędzia platformy Azure dla programu Visual Studio 2015

Najszybszym i najprostszym sposobem instalowania **zestawu Azure SDK dla programu Visual studio 2015** i **Service Fabric SDK oraz narzędzi dla programu Visual Studio 2015** jest użycie [Instalatora platformy sieci Web](https://www.microsoft.com/web/downloads/platform.aspx). Instalator platformy Microsoft Web to bezpłatne narzędzie usprawniające pobieranie, Instalowanie i aktualizowanie niektórych składników platformy sieci Web firmy Microsoft, w tym narzędzi platformy Azure dla programu Visual Studio 2015. Te zestawy SDK można także pobrać i zainstalować jako pojedyncze składniki ze [strony plików do pobrania platformy Azure](https://azure.microsoft.com/downloads/).

## <a name="using-the-web-platform-installer"></a>Korzystanie z Instalatora platformy sieci Web

1. Pobierz i uruchom program [inicjujący Instalatora platformy sieci Web](https://www.microsoft.com/web/handlers/webpi.ashx?command=getinstallerredirect&appid=VWDOrVs2015AzurePack;MicrosoftAzure-ServiceFabric-VS2015).

2. Program inicjujący zainstaluje Instalatora platformy sieci Web (w razie potrzeby) i automatycznie umieści najnowsze wersje **zestawu Azure SDK dla programu Visual studio 2015** oraz **Service Fabric zestaw SDK i narzędzia dla programu Visual Studio 2015** elementy na liście *elementów do zainstalowania* . Kliknij przycisk **Instaluj**.

    ![Instalator platformy sieci Web](../media/sdk/vs2015-install/webpi.png)

3. Na następnym ekranie kliknij pozycję **Akceptuję**. W sieci Web PI rozpocznie się pobieranie i Instalowanie wybranych składników.

4. Po zakończeniu instalacji zostanie wyświetlony ekran potwierdzenia. Kliknij przycisk **Zakończ**. Możesz teraz zamknąć Instalatora platformy sieci Web.

## <a name="verifying-the-installation"></a>Weryfikowanie instalacji

1. W programie Visual Studio 2015 kliknij menu **Narzędzia** , a następnie kliknij pozycję **rozszerzenia i aktualizacje..**..

2. Wyświetlana lista będzie zawierać kilka narzędzi platformy Azure, takich jak **Microsoft Azure App Service Tools**, **Microsoft Azure Storage Connected Service**i **Service Fabric Tools**.

    ![Rozszerzenia i aktualizacje](../media/sdk/vs2015-install/ext-tools.png)
