# <a name="nullable-parameters-must-specify-a-type"></a>Parametry możliwe do wyzernia do wartości muszą określać typ
Parametr zawiera modyfikator typu nullable (?), ale nie określa typu.  
  
 **Identyfikator błędu:** Bc36632  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Określ typ podczas deklarowania parametru typu wartości null.
