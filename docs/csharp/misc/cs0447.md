<span data-ttu-id="1e3ff-103">Nie można używać atrybutów w argumentach typu, tylko w parametrach typu</span><span class="sxs-lookup"><span data-stu-id="1e3ff-103">Attributes cannot be used on type arguments, only on type parameters</span></span>
