# <a name="xml-axis-properties-do-not-support-late-binding"></a>Właściwości osi XML nie obsługują późnego powiązania
Właściwości osi XML został przywołany dla obiekt bez typu.  
  
 **Identyfikator błędu:** BC31168  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że obiekt jest silną kontrolą typów <xref:System.Xml.Linq.XElement> obiektu przed odwołuje się do właściwości osi XML.
