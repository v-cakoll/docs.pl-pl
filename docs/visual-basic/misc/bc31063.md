# <a name="set-method-cannot-have-more-than-one-parameter"></a>Metoda "Set" nie może mieć więcej niż jeden parametr
Więcej niż jeden parametr jest zdefiniowana dla właściwości `Set` procedur.  
  
 **Identyfikator błędu:** BC31063  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń wszelkie dodatkowe parametry, z właściwości `Set` procedur.
