# <a name="argument-39ltargumentnamegt39-cannot-be-a-multidimensional-array"></a>Argument &#39; &lt;argumentname&gt; &#39; nie może być tablicy wielowymiarowej
Pozycja argumentu tablicy jest większa niż 1, ale może być tylko tablicą jednowymiarową.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień pozycja argumentu na 1.
