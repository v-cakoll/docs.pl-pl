## <a name="example"></a><span data-ttu-id="de8df-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="de8df-108">Example</span></span>  

<span data-ttu-id="de8df-109">Poniższy C++ przykład kompiluje do pliku. dll.</span><span class="sxs-lookup"><span data-stu-id="de8df-109">The following C++ example compiles to a .dll file.</span></span> <span data-ttu-id="de8df-110">Zwróć uwagę, że atrybut [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) jest oznaczony jako komentarz w celu wygenerowania błędu.</span><span class="sxs-lookup"><span data-stu-id="de8df-110">Note that the the [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) attribute is commented out in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  

<span data-ttu-id="de8df-111">Poniższy C# przykład wywołuje plik. dll.</span><span class="sxs-lookup"><span data-stu-id="de8df-111">The following C# example calls the .dll file.</span></span> <span data-ttu-id="de8df-112">Próbuje uzyskać dostęp do klasy za pośrednictwem indeksatora, ale ponieważ żaden element członkowski nie został zadeklarowany jako domyślny indeksator, zostanie wygenerowany błąd.</span><span class="sxs-lookup"><span data-stu-id="de8df-112">It attempts to access the class via an indexer, but because no member was declared as the default indexer, the error is generated.</span></span> <span data-ttu-id="de8df-113">Możesz rozwiązać ten problem, usuwając komentarz do wiersza `[DefaultMember("myItem")]`.</span><span class="sxs-lookup"><span data-stu-id="de8df-113">You can address the error by uncommenting the line `[DefaultMember("myItem")]`.</span></span>
  
```csharp  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```
