# <a name="compiler-error-cs1627"></a><span data-ttu-id="c39ec-102">CS1627 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="c39ec-102">Compiler Error CS1627</span></span>
<span data-ttu-id="c39ec-103">Oczekiwano wyrażenia po instrukcji yield return</span><span class="sxs-lookup"><span data-stu-id="c39ec-103">Expression expected after yield return</span></span>  
  
 <span data-ttu-id="c39ec-104">Ten błąd występuje, gdy `yield` jest używana bez wyrażenia.</span><span class="sxs-lookup"><span data-stu-id="c39ec-104">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="c39ec-105">Aby uniknąć tego błędu, wstaw odpowiedni wyrażenie w instrukcji.</span><span class="sxs-lookup"><span data-stu-id="c39ec-105">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="c39ec-106">Poniższy przykład generuje CS1627:</span><span class="sxs-lookup"><span data-stu-id="c39ec-106">The following sample generates CS1627:</span></span>  
  
```  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
