 <span data-ttu-id="1cf59-107">Poniższy przykład generuje CS0109:</span><span class="sxs-lookup"><span data-stu-id="1cf59-107">The following sample generates CS0109:</span></span>  
  
```csharp  
// CS0109.cs  
// compile with: /W:4  
namespace x  
{  
   public class a  
   {  
      public int i;  
   }  
  
   public class b : a  
   {  
      public new int i;  
      public new int j;   // CS0109  
      public static void Main()  
      {  
      }  
   }  
}  
```
