# <a name="compiler-error-cs1715"></a>CS1715 błąd kompilatora
'Typ1': typ musi być 'Typ2' do dopasowania zgodnym z przesłoniętą składową "MemberName"  
  
 Ten błąd jest taka sama jak [CS0508 błąd kompilatora](../../csharp/misc/cs0508.md), chyba że CS0508 teraz ma zastosowanie tylko do metody, które mają typy zwracane, gdy CS1715 ma zastosowanie do właściwości i indeksatorów, którzy mają tylko "typy" zamiast "typy zwracane".  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje CS1715.  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
