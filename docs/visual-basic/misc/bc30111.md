# <a name="lttypenamegt-is-an-interface-type-and-cannot-be-used-as-an-expression"></a>"&lt;typename&gt;" jest typem interfejsu i nie można używać jako wyrażenia
Nazwa typu, który występuje, gdy wyrażenie jest wymagana. Wyrażenie musi składać się z kombinacji zmienne, stałe, literały, właściwości i `Function` wywołań procedur.  
  
 **Identyfikator błędu:** BC30111  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nazwę typu, a wyrażenie za pomocą elementów prawidłowe.
