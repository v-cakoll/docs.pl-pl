# <a name="compiler-error-cs1512"></a>CS1512 błąd kompilatora
Słowo kluczowe "base" nie jest dostępna w bieżącym kontekście  
  
 [Podstawowej](../../csharp/language-reference/keywords/base.md) użyto słowa kluczowego poza metoda, właściwość lub konstruktora.  
  
 Poniższy przykład generuje CS1512:  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
