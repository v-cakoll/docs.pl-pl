# <a name="icorthreadpool-interface"></a>ICorThreadpool — Interfejs
Udostępnia metody do uzyskiwania dostępu do puli wątków.  
  
> [!NOTE]
>  Ten interfejs jest zarezerwowana tylko do użytku wewnętrznego.
