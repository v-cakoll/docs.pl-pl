# <a name="compiler-warning-level-1-cs3011"></a>Kompilator CS3011 ostrzegawcze (poziom 1)
"członek": tylko elementy członkowskie zgodne ze specyfikacją CLS mogą być abstrakcyjne  
  
 Element członkowski klasy nie może jednocześnie być [abstrakcyjny](../../csharp/language-reference/keywords/abstract.md) i niezgodna z typowych specyfikacji języka (CLS). Ze specyfikacją CLS Określa, że wykonywane są wszystkie elementy członkowskie klasy. Aby uzyskać więcej informacji dotyczących zgodności ze specyfikacją CLS, zobacz [pisanie kodu zgodne ze specyfikacją CLS](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) i [niezależność od języka i elementy niezależne od języka](https://msdn.microsoft.com/library/12a7a7h3).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS3011:  
  
```  
// CS3011.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public abstract class I  
{  
    [CLSCompliant(false)]  
    public abstract int M();   // CS3011  
  
    // OK  
    [CLSCompliant(false)]  
    public void M2()  
    {  
    }  
}  
  
public class C : I  
{  
    public override int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
