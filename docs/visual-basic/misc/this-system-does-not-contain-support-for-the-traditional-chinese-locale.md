# <a name="this-system-does-not-contain-support-for-the-traditional-chinese-locale"></a>Ten system nie obsługuje dla ustawień regionalnych chiński tradycyjny
Podjęto próbę użycia `VbStrConv.TraditionalChinese` wyliczeniowego, ale system nie obsługuje ustawień regionalnych języka chińskiego tradycyjnego.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Uaktualnij maszynę do obsługi tradycyjnego języka chińskiego ustawień regionalnych lub uruchom aplikację na komputerze, który obsługuje ustawień regionalnych.
