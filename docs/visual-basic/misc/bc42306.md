# <a name="xml-comment-tag-tag-is-not-permitted-on-a-element-language-element"></a>Tag komentarza XML " \<tag> " jest niedozwolony w \<element> elemencie języka ""
Tag komentarza XML \<tag> nie jest dozwolony w \<element> elemencie "". Komentarz XML zostanie zignorowany.  
  
 Tag komentarza XML nie jest obsługiwany przez element języka, do którego jest stosowany.  
  
 **Identyfikator błędu:** BC42306  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń nieobsługiwany Tag komentarza XML.
