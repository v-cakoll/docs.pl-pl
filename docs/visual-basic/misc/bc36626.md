# <a name="lambda-expression-cannot-be-converted-to-typename-because-type-typename-is-declared-mustinherit-and-cannot-be-created"></a>Wyrażenia lambda nie można przekonwertować na element " \<typename> ", ponieważ typ " \<typename> " jest zadeklarowany jako "MustInherit" i nie można go utworzyć
Instrukcja próbuje skonwertować wyrażenie lambda na typ, który może być tylko klasą bazową i nie może służyć do tworzenia wystąpienia.  
  
 **Identyfikator błędu:** BC36626
