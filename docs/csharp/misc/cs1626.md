# <a name="compiler-error-cs1626"></a><span data-ttu-id="26810-102">CS1626 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="26810-102">Compiler Error CS1626</span></span>
<span data-ttu-id="26810-103">Nie można użyć instrukcji yield z wartością w treści bloku try z klauzulą catch</span><span class="sxs-lookup"><span data-stu-id="26810-103">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="26810-104">Instrukcji yield nie jest dozwolone w bloku try, jeśli jest skojarzony z bloku try klauzuli catch.</span><span class="sxs-lookup"><span data-stu-id="26810-104">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="26810-105">Aby uniknąć tego błędu, należy przenieść poza klauzuli catch instrukcji yield.</span><span class="sxs-lookup"><span data-stu-id="26810-105">To avoid this error, move the yield statement out of the catch clause.</span></span>  
  
 <span data-ttu-id="26810-106">Poniższy przykład generuje CS1626:</span><span class="sxs-lookup"><span data-stu-id="26810-106">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
