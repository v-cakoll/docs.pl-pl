 <span data-ttu-id="f4c4b-104">Jeśli wyrażenie lambda jest konwertowany na drzewo wyrażenia, drzewa wyrażeń musi mieć typ delegata dla jego argumentu.</span><span class="sxs-lookup"><span data-stu-id="f4c4b-104">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="f4c4b-105">Ponadto Wyrażenie lambda musi być konwertowany na typ delegata.</span><span class="sxs-lookup"><span data-stu-id="f4c4b-105">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>

## <a name="example"></a><span data-ttu-id="f4c4b-108">Przykład</span><span class="sxs-lookup"><span data-stu-id="f4c4b-108">Example</span></span>  
 <span data-ttu-id="f4c4b-109">Poniższy przykład generuje CS0835:</span><span class="sxs-lookup"><span data-stu-id="f4c4b-109">The following example generates CS0835:</span></span>  
  
```csharp  
// cs0835.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
