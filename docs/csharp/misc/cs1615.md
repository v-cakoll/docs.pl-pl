# <a name="compiler-error-cs1615"></a><span data-ttu-id="c179d-102">CS1615 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="c179d-102">Compiler Error CS1615</span></span>
<span data-ttu-id="c179d-103">Argument "number" nie powinien być przekazywany ze słowem kluczowym "— słowo kluczowe"</span><span class="sxs-lookup"><span data-stu-id="c179d-103">Argument 'number' should not be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="c179d-104">Słowa kluczowe `ref` lub **się** została użyta, gdy funkcja nie podjęła `ref` lub **się** parametr dla tego argumentu.</span><span class="sxs-lookup"><span data-stu-id="c179d-104">One of the keywords `ref` or **out** was used when the function did not take a `ref` or **out** parameter for that argument.</span></span> <span data-ttu-id="c179d-105">Aby rozwiązać ten problem, Usuń nieprawidłowe słowo kluczowe, a następnie użyj słowa kluczowego odpowiednie, który odpowiada deklaracji funkcji, jeśli istnieje.</span><span class="sxs-lookup"><span data-stu-id="c179d-105">To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.</span></span>  
  
 <span data-ttu-id="c179d-106">Poniższy przykład spowoduje wygenerowanie CS1615:</span><span class="sxs-lookup"><span data-stu-id="c179d-106">The following sample generates CS1615:</span></span>  
  
```csharp  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```
