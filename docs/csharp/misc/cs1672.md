# <a name="compiler-error-cs1672"></a>CS1672 błąd kompilatora
Nieprawidłowa opcja "option" dla opcji/platform; wymagana jest wartość anycpu, x86, Itanium lub x64  
  
 Opcje określić typu procesora; Użyj jednej z wymienionych formularzy.
