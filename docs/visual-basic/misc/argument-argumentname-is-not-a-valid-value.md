# <a name="argument-39ltargumentnamegt39-is-not-a-valid-value"></a>Argument &#39; &lt;argumentname&gt;&#39; nie jest prawidłową wartością
Wartość argumentu jest nieprawidłowa.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Sprawdź wartość argumentu.  
  
2.  Jeśli argument jest dostarczana przez obliczeń, sprawdź wartości, które przetwarza obliczeń.
