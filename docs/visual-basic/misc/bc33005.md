# <a name="39end-operator39-expected"></a>&#39;End Operator&#39; oczekiwano
`End` Instrukcja musi odpowiadać bloku zostaje zakończone. `Operator` Definicja procedury nie został prawidłowo zakończony.  
  
 **Identyfikator błędu:** BC33005  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Dodaj `End Operator` instrukcji na końcu procedury.
