# <a name="compiler-error-cs0447"></a>CS0447 błąd kompilatora
Nie można używać atrybutów w argumentach typu, tylko w parametrach typu  
  
 Ten błąd występuje, gdy zastosować atrybut do argumentem typu, który występuje w instrukcji wywołania. Jest dopuszczalne, aby zastosować atrybut do parametru typu w instrukcji deklaracji klasy lub metody takie jak następujące:  
  
```csharp  
class C<[some attribute] T> {…}  
```  
  
 Następujący wiersz kodu, spowoduje wygenerowanie tego błędu. Zakłada się, że klasa `C`zdefiniowaną w poprzednim wierszu kodu, ma metodę statyczną o nazwie `MyStaticMethod`.  
  
```csharp  
C<[some attribute] T>.MyStaticMethod();  
```  
  
## <a name="example"></a>Przykład  
 Poniższy kod generuje błąd CS0447.  
  
```csharp  
// CS0447.cs  
using System;  
namespace Test41  
{  
    public interface I<A>   
    {  
        void Meth<B>();  
    }  
    public class B : I<int>   
    {  
        void I<[Test] int>.Meth<X>() { }  // CS0447  
    }  
}  
```
