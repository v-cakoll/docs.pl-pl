# <a name="compiler-error-cs0594"></a>Compiler Error CS0594
Stała zmiennoprzecinkowa jest spoza zakresu typu "type"  
  
 Wartość została przypisana do zmiennej zmiennoprzecinkową, która jest zbyt duży dla zmiennych tego typu danych. Zobacz [Tabela typów całkowitych](../../csharp/language-reference/keywords/integral-types-table.md) Aby uzyskać informacje na temat zakresu wartości dozwolone w typach danych.  
  
 Poniższy przykład spowoduje wygenerowanie CS0594:  
  
```csharp  
// CS0594.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         float f = 6.77777777777E400;   // CS0594, value too large  
      }  
   }  
}  
```
