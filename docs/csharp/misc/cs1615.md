<span data-ttu-id="a1703-103">Argument "number" nie należy przekazywać ze słowem kluczowym "— słowo kluczowe"</span><span class="sxs-lookup"><span data-stu-id="a1703-103">Argument 'number' should not be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="a1703-104">Słowa kluczowe `ref` lub **limit** była używana, gdy funkcja nie zostały `ref` lub **limit** parametr dla tego argumentu.</span><span class="sxs-lookup"><span data-stu-id="a1703-104">One of the keywords `ref` or **out** was used when the function did not take a `ref` or **out** parameter for that argument.</span></span> <span data-ttu-id="a1703-105">Aby rozwiązać ten problem, Usuń nieprawidłowe słowo kluczowe i użyj słowa kluczowego odpowiednie odpowiadający deklaracji funkcji, jeśli istnieje.</span><span class="sxs-lookup"><span data-stu-id="a1703-105">To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.</span></span>  
  
 <span data-ttu-id="a1703-106">Poniższy przykład generuje CS1615:</span><span class="sxs-lookup"><span data-stu-id="a1703-106">The following sample generates CS1615:</span></span>  
  
```  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```
