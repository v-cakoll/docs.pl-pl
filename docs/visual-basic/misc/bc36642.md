```  
' Dim increment1 = Function (n) n + 1  
```  
  
 <span data-ttu-id="c2764-104">Poprzednia deklaracja jest prawidłowa Jeśli typ `n` można wywnioskować.</span><span class="sxs-lookup"><span data-stu-id="c2764-104">The previous declaration is valid if the type of `n` can be inferred.</span></span> <span data-ttu-id="c2764-105">Na przykład w przypadku przypisywania poprzedniego wyrażenia lambda do delegata funkcji `Del`:</span><span class="sxs-lookup"><span data-stu-id="c2764-105">For example, if you are assigning the previous lambda expression to a function delegate, `Del`:</span></span>

 <span data-ttu-id="c2764-106">Teraz typ `n` można wywnioskować na podstawie parametru `p`:</span><span class="sxs-lookup"><span data-stu-id="c2764-106">Now the type of `n` can be inferred from parameter `p`:</span></span>
