 <span data-ttu-id="942e5-104">Kompilator próbuje dopasować typy danych na liście argumentów wywołujących i na liście parametrów przeciążenia tak blisko jak to możliwe.</span><span class="sxs-lookup"><span data-stu-id="942e5-104">The compiler attempts to match the data types in the calling argument list and the overload parameter list as closely as possible.</span></span> <span data-ttu-id="942e5-105">Wymaga konwersji rozszerzającej z każdego jednego z argumentów do odpowiadającego mu parametru, niezależnie od tego, czy przełącznik sprawdzania typu ([instrukcja Option Strict](../language-reference/statements/option-strict-statement.md)) ma wartość `On` lub `Off` .</span><span class="sxs-lookup"><span data-stu-id="942e5-105">It requires a widening conversion from every one of your arguments to its corresponding parameter, whether the type checking switch ([Option Strict Statement](../language-reference/statements/option-strict-statement.md)) is `On` or `Off`.</span></span>  
  
 <span data-ttu-id="942e5-106">Jeśli kompilator odnajdzie więcej niż jedno Przeciążenie spełniające wymaganie rozszerzające, szuka przeciążeń, które jest najbardziej specyficzne dla typów danych argumentów, to oznacza, że wywołuje najmniejszą ilość rozszerzania.</span><span class="sxs-lookup"><span data-stu-id="942e5-106">If the compiler finds more than one overload satisfying the widening requirement, it then looks for the overload that is most specific for the argument data types, that is, that calls for the least amount of widening.</span></span> <span data-ttu-id="942e5-107">Generuje ten komunikat o błędzie, gdy jedno Przeciążenie jest bardziej specyficzne dla typu danych jednego argumentu, podczas gdy inne Przeciążenie jest bardziej specyficzne dla typu danych innego argumentu.</span><span class="sxs-lookup"><span data-stu-id="942e5-107">It generates this error message when one overload is more specific for one argument's data type while another overload is more specific for another argument's data type.</span></span> <span data-ttu-id="942e5-108">Aby uzyskać więcej informacji i zapoznać się z przykładem, zobacz [rozpoznawanie przeciążenia](../programming-guide/language-features/procedures/overload-resolution.md).</span><span class="sxs-lookup"><span data-stu-id="942e5-108">For more information and an example, see [Overload Resolution](../programming-guide/language-features/procedures/overload-resolution.md).</span></span>
