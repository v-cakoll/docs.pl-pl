# <a name="compiler-warning-level-1-cs1709"></a>Kompilatora (poziom 1) ostrzeżenie CS1709
Nazwa pliku określona dla dyrektywy preprocesora jest pusta  
  
 Określono dyrektywy preprocesora, który zawiera nazwę pliku, ale ten plik jest pusty. Aby rozwiązać tego ostrzeżenia, należy umieścić potrzebnej zawartości w pliku.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS1709.  
  
```csharp  
// CS1709.cs  
class Test  
{  
    static void Main()  
    {  
        #pragma checksum "" "{406EA660-64CF-4C82-B6F0-42D48172A799}" ""  // CS1709  
    }  
}  
```
