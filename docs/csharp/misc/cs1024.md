# <a name="compiler-error-cs1024"></a>CS1024 błąd kompilatora
Oczekiwano dyrektywy preprocesora  
  
 Linię rozpoczął się symbolem krzyżyk (#), ale kolejne ciąg nie jest prawidłowym [dyrektywy preprocesora](../../csharp/language-reference/preprocessor-directives/index.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS1024:  
  
```csharp  
// CS1024.cs  
#import System   // CS1024  
```
