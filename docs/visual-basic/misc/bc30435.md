# <a name="members-in-a-structure-cannot-be-declared-ltspecifiergt"></a>Nie można zadeklarować składowych w strukturze "&lt;specyfikator&gt;"
Wykorzystano specyfikator, który jest nieprawidłowy w elemencie członkowskim w ramach `Structure` instrukcji.  
  
 **Identyfikator błędu:** BC30435  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń specyfikator.
