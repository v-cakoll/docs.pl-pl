## <a name="examples"></a><span data-ttu-id="818bd-290">Przykłady</span><span class="sxs-lookup"><span data-stu-id="818bd-290">Examples</span></span>  
 <span data-ttu-id="818bd-291">Następujące polecenie generuje schematu XML z `myFile.xdr` i zapisuje go w bieżącym katalogu.</span><span class="sxs-lookup"><span data-stu-id="818bd-291">The following command generates an XML schema from `myFile.xdr` and saves it to the current directory.</span></span>


```console

```  
  
 <span data-ttu-id="818bd-293">Następujące polecenie generuje zestaw danych, który odpowiada określony schemat w języku C# i zapisuje je `XSDSchemaFile.cs` w bieżącym katalogu.</span><span class="sxs-lookup"><span data-stu-id="818bd-293">The following command generates a data set that corresponds to the specified schema in the C# language and saves it as `XSDSchemaFile.cs` in the current directory.</span></span>
