- <span data-ttu-id="ae0fe-107">**Stanowa automatyczna konfiguracja**.</span><span class="sxs-lookup"><span data-stu-id="ae0fe-107">**Stateful auto-configuration**.</span></span> <span data-ttu-id="ae0fe-108">Ten typ konfiguracji wymaga pewnego poziomu interwencji człowieka, ponieważ wymaga protokołu konfiguracji hosta dynamicznego dla serwera IPv6 (DHCPv6) do instalacji i administrowania węzłami.</span><span class="sxs-lookup"><span data-stu-id="ae0fe-108">This type of configuration requires a certain level of human intervention because it needs a Dynamic Host Configuration Protocol for IPv6 (DHCPv6) server for the installation and administration of the nodes.</span></span> <span data-ttu-id="ae0fe-109">Serwer DHCPv6 przechowuje listę węzłów, do których dostarcza informacje o konfiguracji.</span><span class="sxs-lookup"><span data-stu-id="ae0fe-109">The DHCPv6 server keeps a list of nodes to which it supplies configuration information.</span></span> <span data-ttu-id="ae0fe-110">Przechowuje również informacje o stanie, dzięki czemu serwer wie, jak długo każdy adres jest w użyciu i kiedy może być dostępny do ponownego przypisania.</span><span class="sxs-lookup"><span data-stu-id="ae0fe-110">It also maintains state information so the server knows how long each address is in use, and when it might be available for reassignment.</span></span>

 <span data-ttu-id="ae0fe-115">Niezależnie od sposobu określania adresu węzeł musi sprawdzić, czy jego potencjalny adres jest unikatowy dla łącza lokalnego.</span><span class="sxs-lookup"><span data-stu-id="ae0fe-115">Regardless of how the address is determined, the node must verify that its potential address is unique to the local link.</span></span> <span data-ttu-id="ae0fe-116">Odbywa się to poprzez wysłanie wiadomości na żądanie sąsiada na potencjalny adres.</span><span class="sxs-lookup"><span data-stu-id="ae0fe-116">This is done by sending a neighbor solicitation message to the potential address.</span></span> <span data-ttu-id="ae0fe-117">Jeśli węzeł odbiera odpowiedzi, wie, że adres jest już w użyciu i musi określić inny adres.</span><span class="sxs-lookup"><span data-stu-id="ae0fe-117">If the node receives any response, it knows that the address is already in use and must determine another address.</span></span>
