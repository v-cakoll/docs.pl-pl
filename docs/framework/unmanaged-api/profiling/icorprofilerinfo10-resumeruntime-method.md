# <a name="icorprofilerinfo10resumeruntime-method"></a>ICorProfilerInfo10:: ResumeRuntime, Metoda
  
Wznawia środowisko uruchomieniowe bez wykonywania operacji GC.   
  
## <a name="syntax"></a>Składnia  
  
```cpp
HRESULT ResumeRuntime();
```
