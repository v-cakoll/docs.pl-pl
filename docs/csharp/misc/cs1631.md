# <a name="compiler-error-cs1631"></a>Błąd kompilatora CS1631
Nie można uzyskać wartości w treści klauzuli catch  
  
 Yield Instrukcji nie jest dozwolone z treści catch klauzuli. Aby uniknąć tego błędu, przenieś yield instrukcji poza treści catch klauzuli.  
  
 Poniższy przykład generuje CS1631:  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }
  
   public static void Main()
   {  
   }  
}  
```
