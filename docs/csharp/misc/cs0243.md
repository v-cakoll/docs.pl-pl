<span data-ttu-id="5598b-103">Atrybut Conditional jest nieprawidłowy w elemencie "method", ponieważ jest to metoda przesłonięcia</span><span class="sxs-lookup"><span data-stu-id="5598b-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
 <span data-ttu-id="5598b-104">[Warunkowego](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) atrybut nie jest dozwolony na metodzie, która jest oznaczona za pomocą [zastąpienia](../../csharp/language-reference/keywords/override.md) — słowo kluczowe.</span><span class="sxs-lookup"><span data-stu-id="5598b-104">The [Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="5598b-105">Aby uzyskać więcej informacji, zobacz [, wiedząc, gdy użycie zastępowania i nowych słów kluczowych](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="5598b-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="5598b-106">Kompilator nigdy nie wiąże się Przesłaniaj metody; tylko tworzy powiązanie z metody podstawowej, a środowisko uruchomieniowe języka wspólnego wywołuje override, zgodnie z potrzebami.</span><span class="sxs-lookup"><span data-stu-id="5598b-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="5598b-107">Poniższy przykład spowoduje wygenerowanie CS0243:</span><span class="sxs-lookup"><span data-stu-id="5598b-107">The following sample generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
