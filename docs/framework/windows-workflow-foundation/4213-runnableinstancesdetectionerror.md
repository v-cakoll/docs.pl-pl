# <a name="4213---runnableinstancesdetectionerror"></a>4213 - RunnableInstancesDetectionError
## <a name="properties"></a>Właściwości  
  
|||  
|-|-|  
|ID|4213|  
|Słowa kluczowe|WFInstanceStore|  
|Poziom|Błąd|  
|Kanał|Microsoft-Windows aplikacji debugowania serwera — aplikacje|  
  
## <a name="description"></a>Opis  
 Wykrywanie możliwych do uruchomienia wystąpień nie powiodło się z powodu wyjątku  
  
## <a name="message"></a>Komunikat  
 Wykrywanie możliwych do uruchomienia wystąpień nie powiodło się z powodu następującego wyjątku.

|Domeny aplikacji|xs:String|Długość ciągu zwróconego przez AppDomain.CurrentDomain.FriendlyName.|
