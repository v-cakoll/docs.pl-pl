# <a name="compiler-error-cs1619"></a>CS1619 błąd kompilatora
Nie można utworzyć pliku tymczasowego "filename" — Przyczyna  
  
 Kompilator nie może utworzyć pliku tymczasowego w danym powodu (na przykład dysk jest zapełniony).
