# <a name="compiler-warning-level-3-cs0168"></a>Kompilator CS0168 ostrzegawcze (poziom 3)
Zmienna "var" jest zadeklarowana, ale nigdy używane

Kompilator generuje ostrzeżenie trzech poziom, gdy zadeklarować zmiennej, ale nie należy używać.
  
Poniższy przykład generuje ostrzeżenie CS0168:  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
