# <a name="variables-in-modules-cannot-be-declared-specifier"></a>Zmienne w modułach nie mogą być deklarowane jako " \<specifier> "
Użyto specyfikatora, takiego jak `MustInherit` , na zmiennej w `Module` instrukcji. Nie można tworzyć wystąpień modułów, nie obsługują dziedziczenia ani implementowania interfejsów.  
  
 **Identyfikator błędu:** BC30593  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Usuń specyfikator.
