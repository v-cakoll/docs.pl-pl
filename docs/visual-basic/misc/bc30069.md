# <a name="for-loop-control-variable-ltvariablenamegt-already-in-use-by-an-enclosing-for-loop"></a>Aby uzyskać zmienna sterująca pętli "&lt;nazwa_zmiennej&gt;" już w użyciu przez otaczającą pętlę For
Zagnieżdżone `For` pętli wszystkich musi używać zmiennych unikatową kontroli.  
  
 **Identyfikator błędu:** BC30069  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmiana nazwy zmiennej kontrolki.
