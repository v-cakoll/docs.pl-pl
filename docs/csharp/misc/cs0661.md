# <a name="compiler-warning-level-3-cs0661"></a><span data-ttu-id="5a131-102">Kompilator CS0661 ostrzegawcze (poziom 3)</span><span class="sxs-lookup"><span data-stu-id="5a131-102">Compiler Warning (level 3) CS0661</span></span>
<span data-ttu-id="5a131-103">"class" definiuje operator == lub! =, lecz nie przesłania metody Object.GetHashCode()</span><span class="sxs-lookup"><span data-stu-id="5a131-103">'class' defines operator == or operator != but does not override Object.GetHashCode()</span></span>  
  
 <span data-ttu-id="5a131-104">Kompilator wykryto równości zdefiniowane przez użytkownika lub operator nierówności, ale nie zastąpienie **GetHashCode** funkcji.</span><span class="sxs-lookup"><span data-stu-id="5a131-104">The compiler detected the user-defined equality or inequality operator, but no override for the **GetHashCode** function.</span></span> <span data-ttu-id="5a131-105">Operator równości i nierówności zdefiniowane przez użytkownika wskazuje, czy też chcesz zastąpić **GetHashCode** funkcji.</span><span class="sxs-lookup"><span data-stu-id="5a131-105">A user-defined equality or inequality operator implies that you also want to override the **GetHashCode** function.</span></span>  
  
 <span data-ttu-id="5a131-106">Poniższy przykład generuje CS0661:</span><span class="sxs-lookup"><span data-stu-id="5a131-106">The following sample generates CS0661:</span></span>  
  
```csharp  
// CS0661.cs  
// compile with: /W:3  
class Test   // CS0661  
{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   public static bool operator != (object o, Test t)  
   {  
      return true;  
   }  
  
   public override bool Equals(object o)  
   {  
      return true;  
   }  
  
   // uncomment the GetHashCode function to resolve  
   // public override int GetHashCode()  
   // {  
   //    return 0;  
   // }  
  
   public static void Main()  
   {  
   }  
}  
```
