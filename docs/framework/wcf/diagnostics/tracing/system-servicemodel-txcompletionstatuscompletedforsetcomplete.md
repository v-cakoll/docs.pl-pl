# <a name="systemservicemodeltxcompletionstatuscompletedforsetcomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForSetComplete
Transakcja określona dla określonej operacji została wykonana wskutek wywołania metody SetTransactionComplete.  
  
## <a name="description"></a>Opis  
 Śledzone, jeśli użytkownik zdecydował się jawnie bieżącej transakcji przez obiekt OperationContext.
