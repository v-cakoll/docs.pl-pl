 <span data-ttu-id="c15cf-116">Niezależnie od sposobu ustalenia adresu, węzeł musi sprawdzić, czy jego potencjalny adres jest unikatowy dla linku lokalnego.</span><span class="sxs-lookup"><span data-stu-id="c15cf-116">Regardless of how the address is determined, the node must verify that its potential address is unique to the local link.</span></span> <span data-ttu-id="c15cf-117">Jest to realizowane przez wysłanie komunikatu o zażądaniu sąsiada do potencjalnego adresu.</span><span class="sxs-lookup"><span data-stu-id="c15cf-117">This is done by sending a neighbor solicitation message to the potential address.</span></span> <span data-ttu-id="c15cf-118">Jeśli węzeł odbierze odpowiedź, wie, że adres jest już używany i musi określić inny adres.</span><span class="sxs-lookup"><span data-stu-id="c15cf-118">If the node receives any response, it knows that the address is already in use and must determine another address.</span></span>
