# <a name="compiler-error-cs0601"></a><span data-ttu-id="24fbf-102">CS0601 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="24fbf-102">Compiler Error CS0601</span></span>
<span data-ttu-id="24fbf-103">Atrybut DllImport musi być określony dla metody oznaczonej jako "static" i "extern"</span><span class="sxs-lookup"><span data-stu-id="24fbf-103">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="24fbf-104">`DllImport` Atrybut użyto metody, która nie ma słowa kluczowe dostępu.</span><span class="sxs-lookup"><span data-stu-id="24fbf-104">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="24fbf-105">Poniższy przykład generuje CS0601:</span><span class="sxs-lookup"><span data-stu-id="24fbf-105">The following sample generates CS0601:</span></span>  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
