# <a name="39sub-new39-cannot-be-declared-39overrides39"></a>&#39;Sub New&#39; nie można zadeklarować &#39;zastąpienia&#39;
Konstruktor wskazuje, zastępuje on dziedziczonego konstruktora. Nie można zastąpić konstruktorów.  
  
 **Identyfikator błędu:** BC30283  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Overrides` — słowo kluczowe z `Sub` deklaracji.
