# <a name="compiler-error-cs1520"></a><span data-ttu-id="b7f57-102">CS1520 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="b7f57-102">Compiler Error CS1520</span></span>
<span data-ttu-id="b7f57-103">Metoda musi mieć typ zwracany</span><span class="sxs-lookup"><span data-stu-id="b7f57-103">Method must have a return type</span></span>  
  
 <span data-ttu-id="b7f57-104">Metoda, która jest zadeklarowana w klasy, struktury lub interfejsu musi mieć jawnie określony typ zwracany.</span><span class="sxs-lookup"><span data-stu-id="b7f57-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="b7f57-105">W poniższym przykładzie metoda kwadratowy ma wartość zwracaną [ciąg](../../csharp/language-reference/keywords/string.md):</span><span class="sxs-lookup"><span data-stu-id="b7f57-105">In the following example, the Square method has a return value of [string](../../csharp/language-reference/keywords/string.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="b7f57-106">Poniższy przykład spowoduje wygenerowanie CS1520:</span><span class="sxs-lookup"><span data-stu-id="b7f57-106">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520     
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="b7f57-107">Też ten błąd może wystąpić, gdy wielkość liter w nazwie Konstruktor, który różni się od deklaracji klasy lub struktury, jak w poniższym przykładzie.</span><span class="sxs-lookup"><span data-stu-id="b7f57-107">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="b7f57-108">Ponieważ nazwa jest dokładnie taka sama jak nazwa klasy, kompilator interpretuje słowa kluczowe jako metodę regularne nie konstruktora i generuje błąd:</span><span class="sxs-lookup"><span data-stu-id="b7f57-108">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
