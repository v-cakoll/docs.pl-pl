# <a name="compiler-error-cs0200"></a>CS0200 błąd kompilatora
Właściwość lub indeksator "właściwość" nie można przypisać do — jest tylko do odczytu  
  
 Nastąpiła próba do przypisania wartości do [właściwości](../../csharp/programming-guide/classes-and-structs/using-properties.md), ale właściwość nie ma dostępu set. Rozwiązać problem, dodając dostępu set. Aby uzyskać więcej informacji, zobacz [jak: Deklarowanie właściwości odczyt/zapis oraz](../../csharp/programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0200:  
  
```csharp  
// CS0200.cs  
public class MainClass  
{  
    // private int _mi;  
    int I  
    {  
        get  
        {  
            return 1;  
        }  
  
        // uncomment the set accessor and declaration for _mi  
        /*  
        set  
        {  
            _mi = value;  
        }  
        */  
    }  
  
    public static void Main ()  
    {  
        MainClass II = new MainClass();  
        II.I = 9;   // CS0200  
    }  
}  
```
