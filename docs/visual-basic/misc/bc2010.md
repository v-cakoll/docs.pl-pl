# <a name="compilation-failed-ltmessagegt"></a>Kompilacja nie powiodła się: &lt;wiadomości&gt;
Kompilacja nie powiodła się ze względu na okoliczności określony w komunikacie o błędzie  
  
 **Identyfikator błędu:** BC2010  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź komunikat o błędzie w celu ustalenia dokładne przyczyny błędu.
