# <a name="compiler-error-cs0517"></a>CS0517 błąd kompilatora
"class" ma klasy podstawowej i nie można wywołać konstruktora podstawowego  
  
 CS0517 może wystąpić tylko wtedy, gdy .NET Framework środowisko uruchomieniowe języka wspólnego kompiluje kod źródłowy dla klasy obiektów jest to jedyna klasa, która ma klasy podstawowej.
