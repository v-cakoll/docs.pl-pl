# <a name="statement-cannot-appear-within-a-property-body"></a>Instrukcja nie może wystąpić w treści właściwości
Instrukcja nie może wystąpić w treści właściwości. Założono koniec właściwości.  
  
 Nieoczekiwany instrukcji Napotkano właściwość.  
  
 **Identyfikator błędu:** BC30634  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy Twoje właściwości używa poprawnej składni i kończy z `End``Property` utworzenia.
