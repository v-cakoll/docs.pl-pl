# <a name="compiler-error-cs0157"></a><span data-ttu-id="4bc45-102">CS0157 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="4bc45-102">Compiler Error CS0157</span></span>
<span data-ttu-id="4bc45-103">Formant nie może opuścić treści klauzuli finally</span><span class="sxs-lookup"><span data-stu-id="4bc45-103">Control cannot leave the body of a finally clause</span></span>  
  
 <span data-ttu-id="4bc45-104">Wszystkie instrukcje w [koniec](../../csharp/language-reference/keywords/try-catch-finally.md) musi być wykonywany w klauzuli.</span><span class="sxs-lookup"><span data-stu-id="4bc45-104">All of the statements in a [finally](../../csharp/language-reference/keywords/try-catch-finally.md) clause must execute.</span></span> <span data-ttu-id="4bc45-105">Aby uzyskać więcej informacji, zobacz [instrukcje obsługi wyjątków](../../csharp/language-reference/keywords/exception-handling-statements.md) i [wyjątki i obsługa wyjątków](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="4bc45-105">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="4bc45-106">Poniższy przykład generuje CS0157:</span><span class="sxs-lookup"><span data-stu-id="4bc45-106">The following sample generates CS0157:</span></span>  
  
```csharp  
// CS0157.cs  
using System;  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
