# <a name="argument-not-specified-for-parameter-parametername-of-methodname"></a>Nie określono argumentu dla parametru "\<parametername >" z "\<methodname >"
Nie podano argumentu dla wymaganego parametru.  
  
 **Identyfikator błędu:** BC30455  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Należy podać argument określony parametr.
