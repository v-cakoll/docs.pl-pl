# <a name="option-must-be-followed-by-compare-explicit-infer-or-strict"></a>"Opcji" musi następować "Compare", "Explicit", "Infer" lub "Strict"
Tylko opcje, które można ustawić za pomocą `Option` są instrukcji `Compare`, `Explicit`, `Infer`, i `Strict`.  
  
 **Identyfikator błędu:** BC30206  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy option-słowo kluczowe (`Compare`, `Explicit`, `Infer`, lub `Strict`) jest błędnie wpisana.  
  
-   Sprawdź, czy `Option` instrukcja jest niekompletna.
