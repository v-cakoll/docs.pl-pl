# <a name="compiler-error-cs0471"></a>CS0471 błąd kompilatora
Metoda "name" nie jest metody rodzajowej. Jeśli użytkownik Zamierzana była lista wyrażeń, należy ująć w nawiasy < wyrażenie.  
  
 Ten błąd jest generowany, gdy kod zawiera listę wyrażenia bez nawiasów.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS0471.  
  
```csharp  
// CS0471.cs  
// compile with: /t:library  
class Test  
{  
    public void F(bool x, bool y) {}  
    public void F1()  
    {  
        int a = 1, b = 2, c = 3;  
        F(a<b, c>(3));    // CS0471  
        // To resolve, try the following instead:  
        // F((a<b), c>(3));  
    }  
}  
```
