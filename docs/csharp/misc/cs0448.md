# <a name="compiler-error-cs0448"></a>CS0448 błąd kompilatora
Typ zwracany dla ++ lub--operatora musi być typem zawierającym lub pochodzić od typu zawierającego  
  
 Gdy zastąpisz `++` lub `--` operatorów, muszą zwracać taki sam typ co typ zawierający, lub zwracany typ, który pochodzi od typu zawierającego.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0448.  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0448.  
  
```csharp  
// CS0448_b.cs  
public struct S  
{  
   public static S? operator ++(S s) { return new S(); }   // CS0448  
   public static S? operator --(S s) { return new S(); }   // CS0448  
}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}  
}  
```
