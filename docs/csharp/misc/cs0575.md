# <a name="compiler-error-cs0575"></a>CS0575 błąd kompilatora
Tylko typy klasy mogą zawierać destruktorów  
  
 A [struktury](../../csharp/language-reference/keywords/struct.md) nie może zawierać destruktora.  
  
 Poniższy przykład generuje CS0575:  
  
```csharp  
// CS0575.cs  
namespace x  
{  
   public struct iii  
   {  
      ~iii()   // CS0575  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
