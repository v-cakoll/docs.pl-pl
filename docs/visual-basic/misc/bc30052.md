# <a name="array-exceeds-the-limit-of-32-dimensions"></a>Tablicy przekracza 32 wymiarów
Tablice nie może mieć więcej niż 32 wymiarów.  
  
 **Identyfikator błędu:** BC30052  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmniejsz liczbę wymiarów w deklaracji tablicy.
