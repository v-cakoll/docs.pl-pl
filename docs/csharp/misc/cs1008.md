# <a name="compiler-error-cs1008"></a>Błąd kompilatora CS1008
Oczekiwano typu Byte, bajty, Short, UShort, int, uint, Long lub ULONG  
  
 Niektóre typy danych, takie jak [wyliczenia](../language-reference/builtin-types/enum.md), mogą być deklarowane tylko do przechowywania danych określonego typu.  
  
 Poniższy przykład generuje CS1008:  
  
```csharp  
// CS1008.cs  
abstract public class clx  
{  
   enum splitch : char   // CS1008, char not valid type for enums  
   {  
      x, y, z  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
