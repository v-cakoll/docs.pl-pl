# <a name="some-subkeys-cannot-be-deleted"></a>Nie można usunąć niektórych podkluczy
Została podjęta próba usunięcia klucza rejestru, ale operacja nie powiodła się, ponieważ nie można usunąć niektórych podkluczy. Zazwyczaj jest to spowodowane brakiem uprawnień.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Upewnij się, że masz wystarczające uprawnienia do usuwania określonych podkluczy.
