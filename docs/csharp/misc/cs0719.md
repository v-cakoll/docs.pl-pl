# <a name="compiler-error-cs0719"></a>CS0719 błąd kompilatora
"type": elementy tablicy nie może być typu statycznego  
  
 Tablica typu statycznego nie ma sensu, ponieważ elementy tablicy są wystąpienia, ale nie jest możliwe do tworzenia wystąpień typów statycznych.  
  
 Poniższy przykład spowoduje wygenerowanie CS0719:  
  
```csharp  
// CS0719.cs  
public static class SC  
{  
   public static void F()  
   {  
   }  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC[] sca = new SC[10];  // CS0719  
   }  
}  
```
