# <a name="finally-can-only-appear-once-in-a-try-statement"></a>"Finally" może wystąpić tylko raz w instrukcji "Try"
`Finally` Służy do przeprowadzenia `Try...Catch...Finally` block; dlatego go może wystąpić tylko raz na końcu bloku.  
  
 **Identyfikator błędu:** BC30381  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Zlokalizuj i Usuń zbędną `Finally`.
