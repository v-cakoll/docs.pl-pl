<span data-ttu-id="a8c7d-103">Transakcje umożliwiają grupowanie zestawu akcji lub operacji w pojedynczą niepodzielną jednostkę wykonania.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-103">Transactions provide a way to group a set of actions or operations into a single indivisible unit of execution.</span></span> <span data-ttu-id="a8c7d-104">Transakcja jest kolekcją operacji o następujących właściwościach:</span><span class="sxs-lookup"><span data-stu-id="a8c7d-104">A transaction is a collection of operations with the following properties:</span></span>  
  
- <span data-ttu-id="a8c7d-105">Niepodzielność.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-105">Atomicity.</span></span> <span data-ttu-id="a8c7d-106">Pozwala to zagwarantować, że wszystkie aktualizacje ukończone w ramach określonej transakcji zostaną zatwierdzone i wykonane w sposób trwały lub wszystkie są przerywane i wycofywane do poprzedniego stanu.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-106">This ensures that either all of the updates completed under a specific transaction are committed and made durable or they are all aborted and rolled back to their previous state.</span></span>  
  
- <span data-ttu-id="a8c7d-107">Zachowania.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-107">Consistency.</span></span> <span data-ttu-id="a8c7d-108">Gwarantuje to, że zmiany wprowadzone w ramach transakcji reprezentują transformację z jednego stanu spójnego na inny.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-108">This guarantees that the changes made under a transaction represent a transformation from one consistent state to another.</span></span> <span data-ttu-id="a8c7d-109">Na przykład transakcja, która przenosi pieniądze z konta kontroli na konto oszczędnościowe, nie zmienia kwoty pieniędzy w ogólnym koncie bankowym.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-109">For example, a transaction that transfers money from a checking account to a savings account does not change the amount of money in the overall bank account.</span></span>  
  
- <span data-ttu-id="a8c7d-110">Izolacja.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-110">Isolation.</span></span> <span data-ttu-id="a8c7d-111">Zapobiega to obserwowanie niezatwierdzonych zmian należących do innych współbieżnych transakcji.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-111">This prevents a transaction from observing uncommitted changes belonging to other concurrent transactions.</span></span> <span data-ttu-id="a8c7d-112">Izolacja zapewnia abstrakcję współbieżności, przy jednoczesnym zapewnieniu, że jedna transakcja nie może mieć nieoczekiwanego wpływu na wykonywanie innej transakcji.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-112">Isolation provides an abstraction of concurrency while ensuring one transaction cannot have an unexpected impact on the execution of another transaction.</span></span>  
  
- <span data-ttu-id="a8c7d-113">Wytrzymałości.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-113">Durability.</span></span> <span data-ttu-id="a8c7d-114">Oznacza to, że po zatwierdzeniu aktualizacje zasobów zarządzanych (takich jak rekord bazy danych) będą trwałe w wyniku awarii.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-114">This means that once committed, updates to managed resources (such as a database record) will be persistent in the face of failures.</span></span>  
  
 <span data-ttu-id="a8c7d-115">Windows Communication Foundation (WCF) oferuje bogaty zestaw funkcji, które umożliwiają tworzenie transakcji rozproszonych w aplikacji usługi sieci Web.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-115">Windows Communication Foundation (WCF) provides a rich set of features that enable you to create distributed transactions in your Web service application.</span></span>  
  
 <span data-ttu-id="a8c7d-116">Funkcja WCF implementuje obsługę protokołu WS-AtomicTransaction (WS-AT), który umożliwia aplikacjom WCF przepływ transakcji do aplikacji międzyoperacyjnych, takich jak współdziałające usługi sieci Web stworzone przy użyciu technologii innej firmy.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-116">WCF implements support for the WS-AtomicTransaction (WS-AT) protocol that enables WCF applications to flow transactions to interoperable applications, such as interoperable Web services built using third-party technology.</span></span> <span data-ttu-id="a8c7d-117">Funkcja WCF implementuje również obsługę protokołu transakcji OLE, który może być używany w scenariuszach, w których nie ma potrzeby włączania przepływu transakcji.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-117">WCF also implements support for the OLE Transactions protocol, which can be used in scenarios where you do not need interop functionality to enable transaction flow.</span></span>  
  
 <span data-ttu-id="a8c7d-118">Możesz użyć pliku konfiguracji aplikacji, aby skonfigurować powiązania do włączania lub wyłączania przepływu transakcji, a także ustawić żądany protokół transakcyjny dla powiązania.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-118">You can use an application configuration file to configure bindings to enable or disable transaction flow, as well as set the desired transaction protocol on a binding.</span></span> <span data-ttu-id="a8c7d-119">Ponadto można ustawić limity czasu transakcji na poziomie usługi przy użyciu pliku konfiguracji.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-119">In addition, you can set transaction time-outs at the service level using the configuration file.</span></span> <span data-ttu-id="a8c7d-120">Aby uzyskać więcej informacji, zobacz [Włączanie przepływu transakcji](enabling-transaction-flow.md).</span><span class="sxs-lookup"><span data-stu-id="a8c7d-120">For more information, see [Enabling Transaction Flow](enabling-transaction-flow.md).</span></span>  
  
 <span data-ttu-id="a8c7d-121">Atrybuty transakcji w <xref:System.ServiceModel> przestrzeni nazw umożliwiają wykonywanie następujących czynności:</span><span class="sxs-lookup"><span data-stu-id="a8c7d-121">Transaction attributes in the <xref:System.ServiceModel> namespace allow you to do the following:</span></span>

- <span data-ttu-id="a8c7d-123">Włącz funkcje transakcji i skonfiguruj zachowanie realizacji transakcji przy użyciu <xref:System.ServiceModel.OperationBehaviorAttribute> atrybutu.</span><span class="sxs-lookup"><span data-stu-id="a8c7d-123">Enable transactions functionality and configure transaction completion behavior using the <xref:System.ServiceModel.OperationBehaviorAttribute> attribute.</span></span>
