# <a name="39optional39-expected"></a>&#39;Opcjonalne&#39; oczekiwano
Opcjonalny argument w deklaracji procedury następuje wymaganego argumentu. Każdy argument po opcjonalny argument musi być również opcjonalne.  
  
 **Identyfikator błędu:** BC30202  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
1.  Jeśli argument ma być wymagane, przenieś go do poprzedź pierwszy argument opcjonalne na liście argumentów.  
  
2.  Jeśli argument ma być opcjonalne, użyj `Optional` — słowo kluczowe.
