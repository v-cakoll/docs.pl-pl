# <a name="compiler-error-cs1678"></a><span data-ttu-id="2a333-102">CS1678 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="2a333-102">Compiler Error CS1678</span></span>
<span data-ttu-id="2a333-103">Parametr "number" jest zadeklarowany jako typu "type1", ale powinien być 'type2'</span><span class="sxs-lookup"><span data-stu-id="2a333-103">Parameter 'number' is declared as type 'type1' but should be 'type2'</span></span>  
  
 <span data-ttu-id="2a333-104">Ten błąd występuje, gdy typ parametru w metodę anonimową różni się od deklaracji delegata, który jest rzutowanie metody do.</span><span class="sxs-lookup"><span data-stu-id="2a333-104">This error occurs when the parameter type in an anonymous method is different from the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="2a333-105">Poniższy przykład spowoduje wygenerowanie CS1678:</span><span class="sxs-lookup"><span data-stu-id="2a333-105">The following sample generates CS1678:</span></span>  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors   
{  
   static void Main()   
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
