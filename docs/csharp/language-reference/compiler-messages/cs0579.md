# <a name="compiler-error-cs0579"></a><span data-ttu-id="ca85d-102">Błąd kompilatora CS0579</span><span class="sxs-lookup"><span data-stu-id="ca85d-102">Compiler Error CS0579</span></span>
<span data-ttu-id="ca85d-103">Zduplikowany atrybut "Attribute"</span><span class="sxs-lookup"><span data-stu-id="ca85d-103">Duplicate 'attribute' attribute</span></span>  
  
 <span data-ttu-id="ca85d-104">Nie można określić tego samego atrybutu więcej niż raz, chyba że atrybut określa **AllowMultiple = true** w [AttributeUsage](../attributes/general.md).</span><span class="sxs-lookup"><span data-stu-id="ca85d-104">It is not possible to specify the same attribute more than once unless the attribute specifies **AllowMultiple=true** in its [AttributeUsage](../attributes/general.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="ca85d-105">Przykład</span><span class="sxs-lookup"><span data-stu-id="ca85d-105">Example</span></span>  
 <span data-ttu-id="ca85d-106">Poniższy przykład generuje CS0579.</span><span class="sxs-lookup"><span data-stu-id="ca85d-106">The following example generates CS0579.</span></span>  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
