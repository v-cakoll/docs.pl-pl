# <a name="compiler-error-cs0248"></a>CS0248 błąd kompilatora
Nie można utworzyć tablicy z ujemnym rozmiarem  
  
 Rozmiar tablicy: określono z liczbą ujemną. Aby uzyskać więcej informacji, zobacz [tablic](../../csharp/programming-guide/arrays/index.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS0248:  
  
```csharp  
// CS0248.cs  
class MyClass  
{  
    public static void Main()  
    {  
        int[] myArray = new int[-3] {1,2,3};   // CS0248, pass a nonnegative number  
    }  
}  
```
