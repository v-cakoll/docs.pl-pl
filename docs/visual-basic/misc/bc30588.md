# <a name="omitted-argument-cannot-match-a-paramarray-parameter"></a>Pominięty argument nie pasuje do parametru ParamArray
Pominięto argumentu musi odpowiadać `ParamArray` parametru. `ParamArray`, który umożliwia przekazywanie dowolnej liczby argumentów do procedury, można używać tylko na ostatni argument listy argumentów.  
  
 **Identyfikator błędu:** BC30588  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Podaj pominięty argument.
