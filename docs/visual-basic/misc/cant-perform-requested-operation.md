# <a name="cant-perform-requested-operation-visual-basic"></a>Nie można wykonać żądanej operacji (Visual Basic)
Nie można wykonać operacji, ponieważ spowodowałoby to unieważnienie bieżącego stanu projektu. Można na przykład próbować programowo zmodyfikować kod, który jest uruchomiony.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zatrzymaj wykonywanie kodu, wprowadź modyfikacje w razie potrzeby, a następnie spróbuj ponownie wykonać operację.
