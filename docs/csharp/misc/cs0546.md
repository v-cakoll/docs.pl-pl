 <span data-ttu-id="2f305-104">Próba przesłonięcia jednej z metod dostępu do właściwości nie powiodła się, ponieważ nie można zastąpić metody dostępu.</span><span class="sxs-lookup"><span data-stu-id="2f305-104">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="2f305-105">Ten błąd może wystąpić, jeśli:</span><span class="sxs-lookup"><span data-stu-id="2f305-105">This error can occur if:</span></span>

 <span data-ttu-id="2f305-108">Jeśli nie chcesz przesłonić właściwości klasy bazowej, możesz użyć słowa kluczowego [New](../language-reference/keywords/new-modifier.md) przed właściwością w klasie pochodnej.</span><span class="sxs-lookup"><span data-stu-id="2f305-108">If you do not want to override the base class property, you can use the [new](../language-reference/keywords/new-modifier.md) keyword before the property in derived class.</span></span>

## <a name="example"></a><span data-ttu-id="2f305-110">Przykład</span><span class="sxs-lookup"><span data-stu-id="2f305-110">Example</span></span>  
 <span data-ttu-id="2f305-111">Poniższy przykład generuje CS0546, ponieważ klasa bazowa nie deklaruje metody dostępu set dla właściwości.</span><span class="sxs-lookup"><span data-stu-id="2f305-111">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
