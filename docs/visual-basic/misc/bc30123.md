# <a name="end-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje "End" są nieprawidłowe w oknie bezpośrednim
`Stop` i `End` instrukcje wstrzymał wykonywanie i nie są dozwolone w kontekście debugowania.  
  
 **Identyfikator błędu:** BC30123  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Nie wydają `End` lub `Stop` instrukcji w **bezpośrednie** okna.
