# <a name="compiler-error-cs0053"></a>CS0053 błąd kompilatora
Niespójność dostępności: typ właściwości "type" jest mniej dostępny niż właściwość "property"  
  
 Konstrukcję publicznego musi zwracać obiekt publicznie. Aby uzyskać więcej informacji, zobacz [modyfikatory dostępu](../../csharp/programming-guide/classes-and-structs/access-modifiers.md).  
  
 Poniższy przykład generuje CS0053:  
  
```  
// CS0053.cs  
class MyClass //defaults to private accessibility  
// try the following line instead  
// public class MyClass  
{  
}  
  
public class MyClass2  
{  
   public MyClass myProperty   // CS0053  
   {  
      get  
      {  
         return new MyClass();  
      }  
      set  
      {  
      }  
   }  
}  
  
public class MyClass3  
{  
   public static void Main()  
   {  
   }  
}  
```
