# <a name="compiler-warning-level-3-cs1700"></a>Ostrzeżenie kompilatora (poziom 3) CS1700
Odwołanie do zestawu Nazwa zestawu jest nieprawidłowe i nie można rozpoznać  
  
 To ostrzeżenie wskazuje, że atrybut, taki jak <xref:System.Runtime.CompilerServices.InternalsVisibleToAttribute>, nie został poprawnie określony.  
  
 Aby uzyskać więcej informacji, zobacz [przyjaznych zestawów](../../../standard/assembly/friend-assemblies.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1700.  
  
```csharp  
// CS1700.cs  
// compile with: /target:library  
using System.Runtime.CompilerServices;  
[assembly:InternalsVisibleTo("app2, Retargetable=f")]   // CS1700  
[assembly:InternalsVisibleTo("app2")]   // OK  
```
