# <a name="compiler-error-cs0245"></a>Błąd kompilatora CS0245
Destruktory i obiekty. Nie można wywołać metody Finalize bezpośrednio. Rozważ wywołanie metody IDisposable. Dispose, jeśli jest dostępna.  
  
 Aby uzyskać więcej informacji, zobacz [programowanie podstawowe do wyrzucania elementów](../../standard/garbage-collection/index.md) bezużytecznych i [destruktorów](../programming-guide/classes-and-structs/destructors.md).  
  
 Poniższy przykład generuje CS0245:  
  
```csharp  
// CS0245.cs  
using System;  
using System.Collections;  
  
class MyClass // : IDisposable  
{  
   /*  
   public void Dispose()  
   {  
      // cleanup code goes here  
   }  
   */  
  
   void m()  
   {  
      this.Finalize();   // CS0245  
      // this.Dispose();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
