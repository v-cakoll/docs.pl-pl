# <a name="compiler-error-cs0568"></a>Błąd kompilatora CS0568
Struktury nie mogą zawierać jawnych konstruktorów bez parametrów  
  
 Każda [Struktura](../language-reference/keywords/struct.md) ma już Konstruktor bez parametrów, który inicjuje obiekt na zero. W związku z tym konstruktory, które można utworzyć dla struktury, muszą mieć co najmniej jeden parametr.  
  
 Poniższy przykład generuje CS0568:  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
