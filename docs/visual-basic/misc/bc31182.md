# <a name="attribute-value-is-not-valid-expecting-value1-or-value2"></a>Wartość atrybutu jest nieprawidłowy; Oczekiwano "wartość1" lub "wartość2"
Określono nieprawidłową wartość dla atrybutu literał dokumentu XML. Wartości prawidłowego atrybutu literał dokumentu XML są oparte na specyfikacji XML 1.0.  
  
 **Identyfikator błędu:** BC31182  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień wartość atrybutu na wartość, która jest prawidłowa na Specyfikacja XML 1.0.
