# <a name="constants-must-have-a-value"></a>Stałe muszą mieć wartość
Zadeklarowano stałą bez przypisywania jej wartości.  
  
 **Identyfikator błędu:** BC30438  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Przypisz wartość do stałej.
