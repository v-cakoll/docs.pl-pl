# <a name="compiler-error-cs1554"></a><span data-ttu-id="4a966-102">Błąd kompilatora CS1554</span><span class="sxs-lookup"><span data-stu-id="4a966-102">Compiler Error CS1554</span></span>
<span data-ttu-id="4a966-103">Deklaracja jest nieprawidłowa; \<Użyj typu > operatora op (...) INSTEAD</span><span class="sxs-lookup"><span data-stu-id="4a966-103">Declaration is not valid; use '\<type> operator op (...' instead</span></span>  
  
<span data-ttu-id="4a966-104">Zwracany typ [operatora przeciążonego](../language-reference/operators/operator-overloading.md) musi występować przed `operator` słowem kluczowym.</span><span class="sxs-lookup"><span data-stu-id="4a966-104">The return type of an [overloaded operator](../language-reference/operators/operator-overloading.md) must appear before the `operator` keyword.</span></span>
  
<span data-ttu-id="4a966-105">Poniższy przykład generuje CS1554:</span><span class="sxs-lookup"><span data-stu-id="4a966-105">The following sample generates CS1554:</span></span>  
  
```csharp  
// CS1554.cs  
class MyClass  
{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)  
   {  
      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
