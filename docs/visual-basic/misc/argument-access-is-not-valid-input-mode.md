# <a name="argument-39access39-is-not-valid-valid-values-for-input-mode-are-39openaccessread39-and-39openaccessdefault39"></a>Argument &#39;dostępu&#39; jest nieprawidłowy. Prawidłowymi wartościami dla trybu Input są &#39;OpenAccess.Read&#39; i &#39;OpenAccess.Default&#39;
Wartość dostępu jest nieprawidłowy w `Input` tryb.  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Zmień wartość na jedną `OpenAccess.Read` lub `OpenAccess.Default`.
