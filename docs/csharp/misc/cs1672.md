# <a name="compiler-error-cs1672"></a>CS1672 błąd kompilatora
Nieprawidłowa opcja "opcji" dla opcji/platform; wymagana wartość anycpu, x86, Itanium lub x64  
  
 Opcje określania typu procesora; Użyj jednej z formularzy na liście.
