# <a name="compiler-error-cs0210"></a>CS0210 błąd kompilatora
Należy podać inicjatora w deklaracji instrukcji fixed lub using  
  
 Należy zadeklarować i zainicjować zmiennej w [stałej instrukcji](../../csharp/language-reference/keywords/fixed-statement.md). Aby uzyskać więcej informacji, zobacz [niebezpieczny kod i wskaźniki](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Poniższy przykład generuje CS0210:  
  
```  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 Poniższy przykład generuje również CS0210 ponieważ [za pomocą instrukcji](../../csharp/language-reference/keywords/using-statement.md) ma żadnego inicjatora.  
  
```  
// CS0210b.cs  
  
using System.IO;  
class Test   
{  
   static void Main()   
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))   
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
