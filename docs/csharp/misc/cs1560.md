# <a name="compiler-error-cs1560"></a>Błąd kompilatora CS1560
Określono nieprawidłową nazwę pliku dla dyrektywy preprocesora. Nazwa pliku jest za długa lub nie jest prawidłową nazwą pliku  
  
 Nazwa pliku, która została określona przez [#line](../language-reference/preprocessor-directives/preprocessor-line.md) przekroczyła _MAX_PATH (256 znaków) lub wiersz, w `#line` którym znaleziono więcej niż 2000 znaków.  
  
## <a name="example"></a>Przykład  
 Poniższy przykład generuje CS1560.  
  
```csharp  
// cs1560.cs   
using System;   
class MyClass   
{     
   public static void Main()      
   {        
      Console.WriteLine("Normal line #1.");     
      #line 21 "MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890.txt"   // CS1560  
    }  
}  
```
