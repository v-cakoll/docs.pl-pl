# <a name="loop-statements-are-not-valid-in-the-immediate-window"></a>Instrukcje pętli nie są prawidłowe w oknie bezpośrednim
`Loop`Nie można używać w **Immediate** okna.  
  
 **Identyfikator błędu:** BC30708  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń `Loop` oświadczenia debugowania kodu.
