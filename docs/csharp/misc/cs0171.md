# <a name="compiler-error-cs0171"></a><span data-ttu-id="55a67-102">Błąd kompilatora CS0171</span><span class="sxs-lookup"><span data-stu-id="55a67-102">Compiler Error CS0171</span></span>
<span data-ttu-id="55a67-103">Pole zapasowe dla automatycznie implementowanej właściwości „name” musi być w pełni przypisane przed zwróceniem sterowania obiektowi wywołującemu.</span><span class="sxs-lookup"><span data-stu-id="55a67-103">Backing field for automatically implemented property 'name' must be fully assigned before control is returned to the caller.</span></span> <span data-ttu-id="55a67-104">Rozważ wywołanie konstruktora bez parametrów z inicjatora konstruktora.</span><span class="sxs-lookup"><span data-stu-id="55a67-104">Consider calling the parameterless constructor from a constructor initializer.</span></span>

 <span data-ttu-id="55a67-107">Poniższy przykład generuje CS0171:</span><span class="sxs-lookup"><span data-stu-id="55a67-107">The following sample generates CS0171:</span></span>  
  
```csharp  
// CS0171.cs  
struct MyStruct  
{  
   MyStruct(int initField)   // CS0171  
   {  
      // i = initField;      // uncomment this line to resolve this error  
   }  
   public int i;  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyStruct aStruct = new MyStruct();  
   }  
}  
```
