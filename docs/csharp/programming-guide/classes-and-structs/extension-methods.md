<span data-ttu-id="bdfd1-174">W przypadku implementowania metod rozszerzających dla danego typu należy pamiętać o następujących kwestiach:</span><span class="sxs-lookup"><span data-stu-id="bdfd1-174">If you do implement extension methods for a given type, remember the following points:</span></span>
