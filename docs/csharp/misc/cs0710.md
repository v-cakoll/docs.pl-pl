# <a name="compiler-error-cs0710"></a>CS0710 błąd kompilatora
Klasy statyczne nie mogą mieć konstruktorów wystąpienia.  
  
 Nie można utworzyć wystąpienia klasy statycznej, dlatego jego nie ma potrzeby dla konstruktorów. Aby uniknąć tego błędu, usuwania żadnych konstruktorów klas statycznych lub jeśli na pewno chcesz skonstruować wystąpienia, należy klasy niestatycznej.  
  
 Poniższy przykład spowoduje wygenerowanie CS0710:  
  
```csharp  
// CS0710.cs  
public static class C  
{  
   public C()  // CS0710  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
