# <a name="lib-expected"></a>Oczekiwano "lib"
A `Declare` instrukcja Określa nazwę biblioteki, bez używania `Lib` — słowo kluczowe.  
  
 **Identyfikator błędu:** BC30218  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Dodaj `Lib` — słowo kluczowe przed nazwą biblioteki.
