# <a name="unable-to-parse-xml-lterrorgt"></a>Nie można przeanalizować kodu XML: &lt;błąd&gt;

 **Identyfikator błędu:** BC31023  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Sprawdź, czy kod XML jest poprawnie sformułowany, prawidłowe i że plik nie jest uszkodzony.
