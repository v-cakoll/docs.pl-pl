# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-ltnumbergt-characters"></a>W pełni kwalifikowane nazwy dla typów w przestrzeniach nazw nie może być dłuższa niż &lt;numer&gt; znaków
W pełni kwalifikowana nazwa typu nie może być dłuższa niż określoną liczbę znaków.  
  
 **Identyfikator błędu:** BC30031  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Skróć nazwę typu.
