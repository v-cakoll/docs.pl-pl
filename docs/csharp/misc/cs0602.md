# <a name="compiler-warning-level-1-cs0602"></a>Kompilator CS0602 ostrzegawcze (poziom 1)
Funkcja "old_feature" jest przestarzała. Zamiast tego użyj "new_feature"  
  
 Funkcja języka używana w kodzie (*old_feature*) nadal jest obsługiwany, ale ta obsługa mogą zostać usunięte w przyszłej wersji. Zamiast tego należy użyć składni zalecane (*new_feature*).
