# <a name="compiler-error-cs0110"></a><span data-ttu-id="2ae9d-102">CS0110 błąd kompilatora</span><span class="sxs-lookup"><span data-stu-id="2ae9d-102">Compiler Error CS0110</span></span>
<span data-ttu-id="2ae9d-103">Obliczanie wartości stałej dla elementu "const deklaracją" obejmuje definicję cykliczną</span><span class="sxs-lookup"><span data-stu-id="2ae9d-103">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="2ae9d-104">Deklaracja [const](../../csharp/language-reference/keywords/const.md) zmiennej (`a`) nie może odwoływać się do innej zmiennej const (`b`) również odwołujący się (`a`).</span><span class="sxs-lookup"><span data-stu-id="2ae9d-104">The declaration of a [const](../../csharp/language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="2ae9d-105">Poniższy przykład spowoduje wygenerowanie CS0110:</span><span class="sxs-lookup"><span data-stu-id="2ae9d-105">The following sample generates CS0110:</span></span>  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
