# <a name="compiler-error-cs0056"></a>CS0056 błąd kompilatora
Niespójność dostępności: typ zwracany "type" jest mniej dostępny niż operator 'operator'  
  
 Konstrukcja publiczny musi zwracać obiekt dostępny publicznie. Aby uzyskać więcej informacji, zobacz [Modyfikatory dostępu](../../csharp/programming-guide/classes-and-structs/access-modifiers.md).   
  
 Poniższy przykład spowoduje wygenerowanie CS0056:  
  
```csharp  
// CS0056.cs  
class MyClass  
// try the following line instead  
// public class MyClass  
{  
}  
  
public class A  
{  
   public static implicit operator MyClass(A a)   // CS0056  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
