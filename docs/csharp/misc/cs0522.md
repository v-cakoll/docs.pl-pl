# <a name="compiler-error-cs0522"></a><span data-ttu-id="d9993-102">Błąd kompilatora CS0522</span><span class="sxs-lookup"><span data-stu-id="d9993-102">Compiler Error CS0522</span></span>
<span data-ttu-id="d9993-103">"Konstruktor": struktury nie mogą wywoływać konstruktorów klasy bazowej</span><span class="sxs-lookup"><span data-stu-id="d9993-103">'constructor' : structs cannot call base class constructors</span></span>  
  
 <span data-ttu-id="d9993-104">[Struktura](../language-reference/keywords/struct.md) nie może wywołać konstruktora klasy bazowej; Usuń wywołanie do konstruktora klasy bazowej.</span><span class="sxs-lookup"><span data-stu-id="d9993-104">A [struct](../language-reference/keywords/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="d9993-105">Poniższy przykład generuje CS0522:</span><span class="sxs-lookup"><span data-stu-id="d9993-105">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
