# <a name="evaluation-of-expression-or-statement-stopped"></a>Szacowanie wyrażenia lub instrukcji zatrzymana

 **Identyfikator błędu:** BC30721  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
- Zezwalaj na wykonywanie kontynuować bez zaznaczania **Zatrzymaj obliczanie**.
