# <a name="compiler-error-cs0664"></a><span data-ttu-id="87b62-102">Błąd kompilatora CS0664</span><span class="sxs-lookup"><span data-stu-id="87b62-102">Compiler Error CS0664</span></span>
<span data-ttu-id="87b62-103">Literał typu Double nie może być niejawnie konwertowany na typ "Type"; Użyj sufiksu "sufiks", aby utworzyć literał tego typu</span><span class="sxs-lookup"><span data-stu-id="87b62-103">Literal of type double cannot be implicitly converted to type 'type'; use an 'suffix' suffix to create a literal of this type</span></span>  
  
 <span data-ttu-id="87b62-104">Nie można ukończyć przypisywania; Użyj sufiksu, aby poprawić instrukcję.</span><span class="sxs-lookup"><span data-stu-id="87b62-104">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="87b62-105">Dokumentacja dla każdego typu identyfikuje odpowiedni sufiks dla tego typu.</span><span class="sxs-lookup"><span data-stu-id="87b62-105">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="87b62-106">Aby uzyskać więcej informacji na temat konwersji, zobacz [rzutowanie i konwersje typów](../programming-guide/types/casting-and-type-conversions.md).</span><span class="sxs-lookup"><span data-stu-id="87b62-106">For more information on conversions, see [Casting and Type Conversions](../programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="87b62-107">Poniższy przykład generuje CS0664:</span><span class="sxs-lookup"><span data-stu-id="87b62-107">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
