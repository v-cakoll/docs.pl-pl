# <a name="lttypenamegt-is-a-structure-type-and-cannot-be-used-as-an-expression"></a>"&lt;typename&gt;" jest typ struktury i nie można używać jako wyrażenia
Nazwa typu, który występuje, gdy wyrażenie jest wymagana. Wyrażenie musi zawierać kombinację zmienne, stałe, literały, właściwości i `Function` wywołań procedur.  
  
 **Identyfikator błędu:** BC30110  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń nazwę typu, a wyrażenie za pomocą elementów prawidłowe.
