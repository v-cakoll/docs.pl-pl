# <a name="39custom39-modifier-can-only-be-used-immediately-before-an-39event39-declaration"></a>&#39; Niestandardowe &#39; Modyfikator można używać tylko bezpośrednio przed &#39; Zdarzenie &#39; Deklaracja
`Custom` — Słowo kluczowe musi poprzedzać tylko `Event` — słowo kluczowe w deklaracji zdarzenia niestandardowego.  
  
 **Identyfikator błędu:** BC31140  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Przenieś `Custom` — słowo kluczowe poprzedzającą `Event` — słowo kluczowe.  
  
     —lub—  
  
-   Usuń `Custom` — słowo kluczowe.
