# <a name="method-methodname-is-already-declared-in-interface-interfacename"></a>Metoda "\<methodname >" jest już zadeklarowany w interfejsie "\<interfacename >"
Zadeklarowano metody interfejsu po raz drugi duplikowania ich.  
  
 **Identyfikator błędu:** BC30377  
  
## <a name="to-correct-this-error"></a>Aby poprawić ten błąd  
  
-   Usuń zduplikowane deklarację.
