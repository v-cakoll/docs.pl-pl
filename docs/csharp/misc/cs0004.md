# <a name="compiler-error-cs0004"></a>CS0004 błąd kompilatora
Ostrzeżenie traktowane jako błąd  
  
 Skompilowany przy użyciu `/warnaserror` opcji kompilatora, która powoduje, że kompilator generowanie ostrzeżenia jako błędy.
