# <a name="compiler-warning-level-2-cs0618"></a>Ostrzeżenie kompilatora (poziom 2) CS0618
"członek" jest przestarzały: "tekst"  
  
 Element członkowski klasy został oznaczony za pomocą `Obsolete` atrybut w taki sposób, że gdy odwołuje się do składowej klasy zostanie wygenerowane ostrzeżenie. Aby uzyskać więcej informacji, zobacz [wspólne atrybuty](../../programming-guide/concepts/attributes/common-attributes.md).  
  
 Poniższy przykład spowoduje wygenerowanie CS0618:  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
