# <a name="compiler-error-cs0231"></a><span data-ttu-id="0858c-102">Błąd kompilatora CS0231</span><span class="sxs-lookup"><span data-stu-id="0858c-102">Compiler Error CS0231</span></span>
<span data-ttu-id="0858c-103">Parametr params musi być ostatnim parametrem na liście parametrów formalnych.</span><span class="sxs-lookup"><span data-stu-id="0858c-103">A params parameter must be the last parameter in a formal parameter list.</span></span>  
  
 <span data-ttu-id="0858c-104">Parametr [params](../language-reference/keywords/params.md) obsługuje zmienną liczbę argumentów i musi być po wszystkich innych parametrów.</span><span class="sxs-lookup"><span data-stu-id="0858c-104">The [params](../language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="0858c-105">Aby uzyskać więcej informacji, zobacz [Metody](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="0858c-105">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="0858c-106">Poniższy przykład generuje CS0231:</span><span class="sxs-lookup"><span data-stu-id="0858c-106">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}
  
   static void Main()
   {  
   }  
}  
```
