# <a name="compiler-error-cs1043"></a>CS1043 błąd kompilatora
{lub; oczekiwano  
  
 Metoda dostępu do właściwości zadeklarowano niepoprawnie. Aby uzyskać więcej informacji, zobacz [przy użyciu właściwości](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Przykład  
 Poniższy przykład spowoduje wygenerowanie CS1043.  
  
```csharp  
// CS1043.cs  
// compile with: /target:library  
public class MyClass  
{  
   public int DoSomething  
   {  
      get return 1;   // CS1043  
      set {}  
   }  
  
   // OK  
   public int DoSomething2  
   {  
      get { return 1;}  
   }  
}  
```
